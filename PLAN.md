## 1. コンセプト宣言

**「見えていた場所に、いつか本当に辿り着く」ことを核にした、明るい和風探索アクションRPG。**
（内部ショートハンド: 聖剣伝説 × オープンワールド × テラリア。※「オープンワールド」「完全シームレス」は**外向け文言では使わない**——期待値事故防止、GPTレビュー反映）

- **ピッチ（外向け文案）**: 神話の息づく列島を、徒歩で歩き、船で海を渡り、飛空艇で雲海の上まで巡る、探索重視の2DアクションRPG。地上・海の彼方・雲上都市は別々のステージではなく、ひとつの世界の層として存在する。ミトは父の残した勾玉の謎を追いながら、各地の依代を灯し、帰り道を失った魂たちを祓い還していく。見えていた遠景に、いつか本当に辿り着く——その連続した旅の感覚を大切にした作品。
- **1枚絵フック**: 雲海を突き抜けた天磐船の眼下に、明るい列島と白金の空中都市が広がる瞬間（コンセプトアート①として実現済み）。
- **核となる感情**: 「FF9までは飛空艇で全世界を飛べたのに、FF10からマップジャンプになって世界が狭く感じた」——あの喪失感を取り戻す。**世界の広さの体感は、ワールド数や物量ではなく“移動の連続性”から生まれる。**

## 2. 確定設計判断（ユーザー合意事項・変更には再確認が必要）

1. **骨格**: 聖剣系シームレスアクション戦闘 × 自由探索 × 語られない世界の秘密（テラリア式環境語り）。ストーリーは薄め
2. **世界 = 擬似日本列島**（架空名）。地域は日本の地方イメージ（南の珊瑚の島々=沖縄、雪国=東北/北海道、古都、山岳 等）。「全世界」を謳わない正直なスコープで、全員和装も自然になる
3. **三層構造**: 地上=列島 ／ 船で水平線の光をくぐると**常世の浜**（ニライカナイ的・珊瑚の楽園）／ 飛空艇で雲海の上の**高天原**（空中都市群）。**別ワールドではなく1つの連続世界の層**
4. **移動の連続性**: FF6/FF9方式。マップ選択ジャンプ禁止。ファストトラベルは初回連続移動後のQoLのみ
5. **和の置き方 = コスモロジー基底**: 八百万・鳥居・祓い・依代を世界の魂に。トーンは明るい（大神/サクナヒメ系彩度）
6. **探索**: 攻略順自由・地域レベル帯・**強敵地帯スニークでレア先取り**（FF11式）・「まだ来る場所じゃない」を自分で判断
7. **収集 = アクセサリー（勾玉系）厳選**: 武器は少数固定（グラフィック負荷回避）。勾玉・御守り・数珠・鈴・根付・組紐・御札などの和呪具にサブ特性を持たせて厳選・ビルドの楽しさを集約
8. **景色は明るめ・開放感最優先**: 空中都市と明るい砂浜は必ず見せ場にする
9. **時代・要素方針（2026-06-12確定）**: **無時代の神話日本**（大神方式、特定時代に固定しない）。判断基準＝「戦乱・統治・暗殺の匂いがする要素は除外、祈り・暮らし・祭りの要素は採用」
   - 除外: 侍（武家制度）・忍者・城郭・火縄銃・江戸町人文化・現代都市伝説系の妖怪
   - 採用: 巫女・鳥居・勾玉・祭り・棚田・漁村・ファンタジー和装（貫頭衣/狩衣/巫女装束ミックス）・平安風の雅（美術として借用）
   - **妖怪は古典・民話系の二層構造**: 敵＝穢れた魂が妖の形をとったもの（祓うと還る）／住人＝善き妖・精霊（NPC）。同種が両側に存在しうる（川を守る河童と穢れた河童）
   - **魔法＝祓いの三流派**（クラス制は不採用・スプライト負荷回避）: 結び＝祓い波（主人公の家系）／神楽＝巫女系の支援・範囲浄化／式＝式神召喚（攻撃用投射体・最小FSM）。各地の巫女・陰陽師的NPCから技を学ぶ＝探索報酬
   - **初期武器は「祓い刀」**: 注連縄・紙垂を結った祭具としての刀（侍の記号と切り離した意匠）。スクショ映えとプレイヤー期待値を採る

## 3. 世界設定

### 3.1 世界「八洲（やしま）」（仮称・記紀の「大八洲」より）

かつて神々が往来した列島世界。地上・海の彼方・雲海の上の三層からなり、人々は土地ごとの依代（よりしろ）に神を祀って暮らす。近年、依代の光が弱まり「穢れ」が獣や精霊に宿り始めた。主人公は祓いの力を持つ旅人として、勾玉を集め、列島を巡る。

- **語られない秘密**: なぜ依代が弱まるのか／高天原はなぜ封じられたのか——本筋では説明せず、遺構・風景・NPCの断片会話だけで匂わせる（テラリア/Tunic式）

### 3.2 三層と地域（製品版の全体像）

**【地上層】擬似日本列島**（徒歩→船で拡大）
| 地域（仮称） | モデル | 色彩・明るさ | 役割 |
|---|---|---|---|
| 始まりの里山郷 | 日本の原風景（棚田・鎮守の森） | 若草×桜×陽光（最初の地域・体験版） | チュートリアル〜Lv帯低 |
| 瑠璃の多島海 | 瀬戸内 | 青×白帆×段々畑 | 船入手後の最初の海 |
| 宮乃京 | 京都・奈良 | 朱×白壁×柳 | 中央ハブ都市・商業 |
| 霊峰の裾野 | 富士・信州 | 緑×雲×石畳の登山道 | 高Lv帯・縦の地形 |
| 北雪郷 | 東北・北海道 | 白銀×藍×ランタン（雪の反射で明るく） | 高Lv帯・雪ギミック |
| 黒杉の幽谷 | 山陰の深い杉谷 | 唯一のゴミゴミ暗めゾーン | コントラスト担当 |

**【海原層】常世の浜**（船で水平線の光をくぐる）— 沖縄・ニライカナイ。白砂×ターコイズ×珊瑚の鳥居。永遠の楽園、最も明るい場所。Seiken-5.0 `nereidia.ts`（砂州＋珊瑚＋参道）の発展形

**【天空層】高天原**（飛空艇で雲海を突き抜ける）— 白金の空中都市群「高天ノ京」。水盤・風の神楽・浮島回廊。Seiken-5.0 `arca.ts`（88×72天空都市）の発展形。最終領域

### 3.3 層をまたぐ演出（体験の頂点）

- 地上→常世: 「水平線の光柱をくぐる」（白フラッシュ→BGM転調→砂浜パレットへ）
- 地上→高天原: 「雲海に突入し光が溢れる」（霧→晴れ→白金パレットへ）
- **着域演出は設計の中心**: BGM・パレット・敵グラフィックの三点同時切り替えで「別の土地に来た」を30秒以内に体感させる（審査指摘の反映）

## 4. 探索設計（本作の心臓部）

1. **地平線の引力**: 序盤から遠景parallaxに「雲の中に光る建築物（高天原）」「異様に明るい水平線（常世）」を見せ続ける。「あそこは何だろう」を最初に植え付け、移動手段解放で答え合わせする
2. **移動解放 = 世界の再発見**: 徒歩→船→飛空艇のたびに「見えていたのに行けなかった場所」が開く。桜井「ストレス→解消→成長→進展」を移動能力に適用
3. **レベル帯グラデーションと自由攻略**: 地域ごとにレベル帯はあるが順路は強制しない。強敵は見た目とテレグラフで「格上」が分かる。**敵の視界（sightR）外を縫って強敵地帯の宝箱（レア勾玉）を先取りできる**——死んで覚えるのではなく「忍び込んで得をする」FF11式の楽しさ
   - **スニークを「見える遊び」にする手がかり設計**（GPTレビュー反映）: 格上敵は目・角・霊気の色で危険度を表示／視界に入る直前に敵の顔がこちらを向く予備動作／草むら・岩陰・鳥居の影を「安全地帯」として地形で見せる／宝箱までのルートは最低2本／見つかっても即死ではなく逃げ切れる余地を残す
4. **秘密の密度**: 3画面に1個上限（多すぎると感動が薄れる）。怪しい壁・登り口の見えない足場・干潮で現れる道。発見時は陽光エフェクト＋祓い音SE
5. **寄り道のリズム**: メインルート難度5→寄り道7→その先の秘密3（強敵を越えたら楽になる解消感）。行き止まりは必ずランドマークか報酬で終え、袋小路を作らない
6. **開放感**: 森・山は塊として端に寄せ、中央は開ける。常世は地平線まで見通せる白砂。高天原は雲海越しに下が見える open-sky（`arca.ts` EMPTY+bg手法）

## 5. 戦闘設計

- **基盤**: Seiken-5.0 のフレームデータ戦闘を移植（かまえ→判定→ヒットストップ→キャンセルフレーム、テレグラフ12F以上、ヒットストップ三重演出=エンティティ停止+カメラのみ振動+白フラッシュ）
- **武器は製品版8種（固定・強化制、詳細は Systems.md §1 を正とする）**: 祓い刀（初期・チャージ「縛り祓い」＝結び師の家業が戦闘に宿る）／木槌／御鏡（掲げて霊光線を放つ魔法武器・二段階照射）／薙刀（複数貫通）／和弓（縄矢で橋架け）／式符（式鬼を設置召喚）／舞扇（風・移動撃ち）／錫杖（三流派増幅＋突き鳴らし）。**武器＝道具**（フィールドギミック。鍵にせずご褒美と近道に）
- **体験版は3種**: 祓い刀 → 木槌（ヒビ岩破壊）→ 御鏡（暗所で光）
- **魔法スキル＝祓いの三流派**: 結び（祓い波）を軸に、神楽（支援・範囲浄化）と式（式神召喚・遠隔）を各地のNPCから習得。スキルは習得制でスプライトは主人公1セットのまま（クラス制不採用）。武器「式符」（設置・持続）とスキル「式」（一発大技）は操作レイヤーで分担（Systems.md §7）
- **祓い固有メカニクス「祓い波」**（審査指摘により新設）: 敵を攻撃すると穢れが剥がれて**祓いゲージ**が溜まる。解放すると周囲一帯を浄化する大技（大ヒットストップ＋全画面光演出）。発動の溜め18Fは無防備（リスクとリターン）。**装備した勾玉で祓い波の性質が変わる**（炎の勾玉=炎属性化、水鏡の勾玉=発動後回復、風走りの勾玉=発動後加速…）——勾玉ビルドと戦闘が直結する
- **敵設計**: 穢れを帯びた古典妖怪・精霊を「祓う」（殺すではなく浄化→光になって還る）。明るいトーンと整合。地域ごとに妖怪の種類と行動パターンを替える（雪国＝雪女系、海＝海坊主系、里山＝狐火・唐傘系 等）
- **ボス**: 2フェーズ制（HP50%で咆哮+振動+フラッシュ24F→パターン追加）。boss.ts 骨格流用

## 6. 装備設計（勾玉システム）

- **スロット**: 武器1 ＋ アクセサリー2〜3枠
- **アクセ語彙**: 勾玉・御守り・数珠・鈴・根付・組紐・御札（和の呪具・縁起物で統一。勾玉は三種の神器=コスモロジーと設定直結）
- **データ構造**: `{ id, name, icon, traits: TraitId[], flavor }` の**特性スロット式**。グラフィックはアイコン1枚のみで、データ定義だけで量産可能（AI製作と相性最良）
- **設計原則（FF11式）**: 数値インフレを抑え、**サブ特性の個性で選ばせる**。「攻撃力は低いが被弾してもノックバックしない」「雪原で減速しない」「夜だけ攻撃+」等の“変な効果”を歓迎。序盤の品が最後まで現役になりうる
- **入手**: ユニークは固定配置（危険地帯の奥・隠し場所・スニーク先取り）。**製品版から**地方特性を帯びた汎用品の低確率ドロップを追加（雪国=氷耐性系等、「もう少しこの地方で狩りたい」滞在動機）。体験版は固定のみ

## 7. アートディレクション

- **解像度**: 論理400×224・SS=3（1200×672）。タイル16px論理=HD48px、プレイヤー24px=72px、ボス56px=168px（Seiken-5.0 と同一、パイプライン完全流用）
- **明るさ基準**: 地面タイルの輝度 L≥55% を原則（黒杉の幽谷のみ例外）。純黒床禁止、洞窟も darkness 0.40前後+光源のSFC的薄暗さまで
- **パレット**: 地域ごとに主調色を変える（§3.2 の表）。着域時のパレット一変が「旅した感」を作る
- **Asset Forge 生成方針**: 4×4遷移タイルセット（autotrim→四辺8%インセット→BOX縮小）、キャラは「約3頭身・頭=全高1/3・STRICT SHEET LAYOUT」、右向きは左向き反転、攻撃シートは attack モード（中央値均一スケール）——アンチパターン集 2d-rpg-antipatterns.md を全面適用
- **遠景**: HD-2D parallax 背景（近景0.7/遠景0.35 の2枚重ね）で高天原の光・常世の水平線を常時見せる

## 10. 技術アーキテクチャと Phase 0 検証項目

- **基盤**: TypeScript + Vite + 自作Canvasエンジン（Seiken-5.0 から core/gfx/field/combat/scenes/data を移植）。vitest + dev-browser ヘッドレス検証（`window.__yashima` フック）
- **ワールド構造**: 巨大1枚タイルマップへの全面移行は**しない**（審査全員一致の警告）。地域=FieldScene（既存パターン）＋**飛行時は同一世界の縮小表示**を目指す
- **Phase 0 で検証すること（体験版着手前に白黒つける）**:
  1. **飛行カメラ方式**: 飛行フラグON時に地上座標を引き継ぎカメラをズームアウト（論理400×224を1600×896相当に見せる倍率変換）できるか。**不可なら**フォールバック=独立 WorldMapScene＋雲海突入演出（白フラッシュ20F）で遷移感をマスク
  2. **parallax 2枚重ね遠景**: 現行 drawBackground は1シートのみ→2枚重ね拡張＋「草原から認識できるサイズの高天原シルエット」をサンドボックスで実証
  3. **祓い波プロトタイプ**: 戦闘コア移植＋穢れゲージ＋祓い波を敵1種・1画面で動かし「気持ちいい」を確認（グラフィックはプレースホルダー。Opus4.8 の「グラフィック先行で遊べない」教訓）
  4. 多光源 darknessOverlay＋装飾散布の同時駆動 fps 計測
  5. **御鏡の照射プロトタイプ**（体験版武器）: 構え状態＋直線ビーム判定＋曇り鏡石の反射判定＋二段階式（移動中減衰）の実装可否。木槌の重打フレームの手触りも同時に検証（Systems.md §7 参照）
- **Phase 0 合格条件**（技術検証ではなく**体験検証**として判定する——GPTレビュー反映）:
  - 飛行（または WorldMapScene 遷移）で「同じ世界を飛んでいる」と感じられる
  - 草原マップから高天原のシルエットを認識できる
  - 祓い波の発動が通常攻撃より明確に気持ちいい
  - 60fps 基準、最低でも安定45fpsを下回らない
  - スクショ1枚で「明るい和風列島ARPG」と分かる
- **過去の失敗教訓の適用**: 戦闘コアを最初に遊べる状態に／重要修正は1つずつ typecheck・test で裏取り／「完成」報告前にユーザー目線スクショ1周

## 12. 市場ポジショニングとタイトル候補

- **ポジション（GPTレビュー反映・「供給空白」と言い切らない表現に整理）**: 近年、Tunic／Minishoot' Adventures／Mina the Hollower など「探索・秘密・クラシックな手触り」を持つ小〜中規模タイトルには明確な需要がある。本作はそこに「明るい和風列島」「船と飛空艇による連続移動」「祓いと勾玉ビルド」を掛け合わせ、単なるレトロARPGではなく**“旅の広がり”**を前面に出す
- **根拠の区分け**: ①同ジャンルの需要実証＝Mina the Hollower（初週50万本報道）・Tunic（91%好評）・Minishoot'（98%好評）／②日本的モチーフの海外受容＝サクナヒメ（公式100万本）・8番出口（200万本報道）——②はジャンル成功例ではなく「和が通じる」材料として扱う
- **ストア第一文（案）**: 「飛空艇で、雲の上まで。──歩いて、渡って、飛んで巡る、神話の列島を旅する探索アクションRPG」（※「オープンワールド」は使わない）
- **タイトル: 『玉結び（たまむすび）』に決定**（2026-06-12 ユーザー選定）
  - **タイトル自体が二層設計**: 表の意味＝「勾玉を集めて結ぶ」（ゲームの見た目）／裏の意味＝「魂結び」（さまよう魂を結び留める実在の民俗儀式＝物語の真相）。クリア後にタイトルの意味が変わって見える
  - 語源系譜: 産霊（ムスヒ、造化三神タカミムスビ・カミムスビ）→ 結び師（ミトの家業）→ 縛り祓い → 玉結び。古事記由来の要望を満たす

  - **衝突チェック（2026-06-12 簡易実施）**: Steamに「玉結び」のゲームは無し（近似は和風ホラーADV「お結び/OMUSUBI」——名称・ジャンルとも別物）。称呼「タマムスビ」の商標登録が他業種に複数存在（ミックウェア社・食品系等）＋TBSラジオ「たまむすび」（2023終了）の知名度あり。**ストア公開前に J-PlatPat で区分9類・41類（ゲームソフト）の正式確認を行うこと**（TODO）
