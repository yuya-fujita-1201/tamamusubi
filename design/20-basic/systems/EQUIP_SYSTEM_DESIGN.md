## 0. 役割と責務

本書「EQUIP_SYSTEM_DESIGN」は **装備の構造と見せ方** を担う設計書である。

| 設計書 | 担う責務 |
|---|---|
| **EQUIP_SYSTEM_DESIGN（本書）** | 装備スロット構成・武器強化木の画面上の見せ方・鍛冶屋UIの操作フロー・強化失敗の有無・勾玉スロットの運用UI・衣スロットの選択UI・ハクスラ排除の実装指針 |
| **[GROWTH_DESIGN](./GROWTH_DESIGN.md)** | 銭の経済バランス・強化コストの増減カーブ・地域別の想定成長到達点（**銭額・特産IDの実数値はGROWTHが一次情報**） |
| **COMBAT_DESIGN** | フレームデータ・祓い波ゲージの消費/蓄積の実数値・状態異常8系統の詳細定義 |
| **[WEAPON_LIST](../../10-lists/WEAPON_LIST.md)** | 8系統×10段階の段階ID・段階ごとの付与効果・解放地域の確定値 |

> **GROWTH_DESIGNとの住み分け**: 武器強化は両書が触れる共通テーマだが視座が異なる。本書は「プレイヤーが強化画面でどう操作し何を見るか」の構造。GROWTH は「いつ・いくら銭と特産を使えば強化が進むか」のカーブ。**強化に要る銭の実数・特産IDはGROWTH §2を参照**し、本書では重複して定義しない。

## 1. 装備スロット

### 1.1 スロット構成（確定）

```
武器 ×1  ＋  衣 ×1  ＋  勾玉 ×3
```

- 見た目（攻撃モーション・FX）に影響するのは**武器のみ**。衣・勾玉はデータ＋アイコン表示のみ
- **ハクスラはしない**: 武器・衣は固定ラインナップ。同名装備の複数所持なし
- **厳選の楽しさは勾玉に集約**（Systems §0）: 48種（v1.1）の勾玉を3枠に何を入れるかがビルドの核

### 1.2 装備画面の構成

装備画面はポーズメニューからアクセスする1画面。左半分にキャラクターのシルエット、右半分にスロット一覧を表示する。

```
┌────────────────────────────────────────┐
│  [ 武器  ] W1_haraitou 祓い刀 g3 ▶強化  │
│  [ 衣    ] koromo_02 巫女衣         ▶比較  │
│  [ 勾玉1 ] mag_001 杜の勾玉              │
│  [ 勾玉2 ] mag_013 厄除けの御守り         │
│  [ 勾玉3 ] （空）                         │
│                                          │
│  ▼ 所持アイテム一覧（スクロール）          │
└────────────────────────────────────────┘
```

- スロットを選択すると所持リストが展開され、選択・解除ができる
- 武器スロットには「▶強化」ボタンを直接設置し、鍛冶屋UIへのショートカットを提供する（鍛冶屋マップに到着している場合のみ有効）
- 衣スロットには「▶比較」ボタンで現在装備との防御値・環境特性の差分を即時表示する
- 勾玉3枠に枠ごとの制限はなし（どのスロットにどの勾玉を入れても良い）

### 1.3 体験版スロット解放タイミング

| Phase | スロット状態 |
|---|---|
| Phase 1A（15〜20分） | 武器・衣・勾玉3枠すべて開放。勾玉は固定3個（mag_001・mag_013・スニーク先取り枠） |
| Phase 1B（30〜45分） | 上記に加え勾玉が最大8個まで増加。「どれを装備するか」選ばせる |

## 2. 武器強化木の構造

### 2.1 全体像

8系統（W1〜W8）それぞれが **g1（入手時の基本形）→ g10（最終形）** の10段階を持つ線形強化木。分岐なし。各段階は「段階名・攻撃倍率・付与効果の深化点」で構成される。

```
W1_haraitou:
  g1 祓い刀 ──▶ g2 鍛え祓刀 ──▶ g3 里桜祓刀 ──▶ … ──▶ g10 天結祓刀
  [入手時]       [銭+山鉄]       [銭+山鉄×多]             [銭+神鉄（希少）]
```

- **分岐なし・合計80段階**（8×10）。深さで量産し、ビルドの差は勾玉3枠と衣で作る
- 段階解放の条件構造: 「直前の段階を所持していること ＋ 銭 ＋ 地域特産1〜2種」
- 早期入手武器（祓い刀・木槌・御鏡）も **g10 には終盤地域（R10）の希少特産**が必要なため、最後まで強化の伸びしろが残る
- **攻撃倍率の成長幅: g1→g10 で +45〜60%**（数値インフレなし）。差別化は役割の直交性と付与効果の深化で作る

> 銭の実数値・特産の正式IDは [GROWTH_DESIGN §2.2〜2.5](./GROWTH_DESIGN.md) を参照。

### 2.2 チャージ技の段階進化

各武器のチャージ技は、強化段階に応じて**性能が3段階に進化**する。

| 進化帯 | 段階 | 進化の方向性 |
|---|---|---|
| 基本形 | g1〜g3 | 入手時のチャージ技そのもの。効果・範囲が基礎値 |
| 強化形 | g4〜g7 | 効果範囲・ヒット数・継続時間が拡大。付与効果（縛り・状態異常）が強化 |
| 最終形 | g8〜g10 | 質的変化。技の見た目・判定が最終形に変化（例: 祓い刀 g10 の縛り祓い＝居合一閃が前方貫通の大判定へ） |

| 武器 | g1 チャージ技 | g10 最終形の変化 |
|---|---|---|
| W1_haraitou | 縛り祓い＝居合の前方一閃（縛り+祓いゲージ加算。全周/回転斬りなし） | 前方貫通の大判定化（縮地踏込延長）＋祓いゲージ2段加算 |
| W2_kizuchi | 杭打ち落とし（前方衝撃波） | 二段衝撃波→地割れ衝撃波（ボスにも短スタン） |
| W3_mikagami | 日輪解放（光弾8発） | 12発→貫通ビーム化・全敵に光弱点小付与 |
| W4_naginata | 貫通閃（直線突進・多体貫通） | 全画面横断・命中数で祓いゲージ加算 |
| W5_wakyu | 遠当て（収束MAXで貫通強矢） | 画面端まで貫通・命中で全体に微弱点露出 |
| W6_shikifu | 百千鳥（展開中の式鬼を一斉爆ぜ） | 式鬼4体・爆ぜが連鎖（隣接敵に伝播） |
| W7_maiougi | 神楽の風（周囲突風・吹き飛ばし） | 竜巻化（吹上＋持続） |
| W8_shakujo | 響鳴（3秒間ゲージ蓄積2倍） | 突き鳴らしが全周大AoE・パーティ全体ゲージ加算 |

### 2.3 鍛冶屋UI操作フロー

```
① 鍛冶屋マップ（例: sato_kajiya）に入室
② 「武器強化」を選択 → 所持武器リストが展開
③ 強化したい武器を選択 → 現在段階と次段階の差分が表示される
④ 必要素材（銭・特産）と所持量が対比表示される
   - 素材が揃っている: 「▶強化する」ボタンが有効
   - 素材が不足している: 「素材不足」と不足分が表示（ボタンは暗転）
⑤ 強化実行 → 演出（打音SE + 火花FX）→ 段階名が更新される
```

- **強化失敗なし**: ロールバック・消費なしの失敗は存在しない。要求素材を持っていれば必ず成功する（グラインドによる損失を排除）
- 複数武器を連続強化できる（メニューを抜けずに次の武器へ移れる）
- 鍛冶屋が未到達の場所での強化は不可（装備画面の「▶強化」ボタンは無効化される）

### 2.4 段階システムのID体系

`W{n}_romaji_g{1..10}` 形式（WEAPON_LIST §0 準拠）。プレイヤー向けの表示名は段階名（例: 「天結祓刀」）を用い、IDはシステム内部でのみ使用する。

## 3. 勾玉システム

### 3.1 特性カテゴリ5区分

勾玉 mag_001〜040 の主効果は以下の5カテゴリに分類される（Systems §4 の定義を本書で正式化）。

| カテゴリ | 説明 | 代表ID |
|---|---|---|
| **攻撃系** | 攻撃倍率・会心・属性ダメ・特定敵への特効など戦闘の火力に寄与する | mag_002（大型敵からの攻撃UP）・mag_037〜040（御札：常時属性付与） |
| **守り系** | 物理防御・ノックバック軽減・状態異常耐性（穢れ/毒/火点き/縛り/眩み/鈍足/呪い/混乱の8系統に対応） | mag_013〜018（御守り：厄除け・清め・火伏せ・雪除け・闇祓い・不動） |
| **移動系** | 移動速度・ステップ無敵・回避距離・居合チャージ短縮など機動力に寄与する | mag_033〜036（組紐：早足・見切り・居合・連ね） |
| **祓い波変化系** | 装備すると**祓い波（浄化大技）の性質を変える**。属性化・発動後回復・発動後加速・範囲拡大・溜め短縮など（本作の核） | mag_004〜012（勾玉：炎/水鏡/風走り/氷柱/雷鳴/広がり/早結び/常世/結び） |
| **探索系** | 銭ドロップ・素材ドロップ・隠し要素の感知・気配察知など探索効率に寄与する。**必須経路の鍵化は絶対禁止** | mag_024〜028（鈴：神楽鈴・千里・埋火・潮鈴・天津）／mag_029〜032（根付：福禄・恵心・山幸・大漁） |

> **祓い波変化系は本作のビルドの核**: 祓い波の属性・挙動が勾玉1個で変わるため、「炎属性化して群体を焼く」「発動後回復を付けて高難度ボスに挑む」など戦術の幅が広がる。この変化が「厳選の楽しさ」の最大の動機になる。

### 3.2 語彙7種とID帯

勾玉は和の呪具・縁起物の語彙で7種に分類され、各語彙がニッチを担当する（ACCESSORY_LIST §0.2 準拠）。

| 語彙 | ID帯 | 件数 | 主なカテゴリ |
|---|---|---|---|
| 勾玉（まがたま） | mag_001〜012 | 12 | 祓い波変化系・基礎ステータス |
| 御守り（おまもり） | mag_013〜018 | 6 | 守り系 |
| 数珠（じゅず） | mag_019〜023 | 5 | MP・スキル消費減・祓い波蓄積 |
| 鈴（すず） | mag_024〜028 | 5 | 探索系（感知・索敵） |
| 根付（ねつけ） | mag_029〜032 | 4 | 探索系（ドロップ・銭・素材） |
| 組紐（くみひも） | mag_033〜036 | 4 | 移動系 |
| 御札（おふだ） | mag_037〜040 | 4 | 攻撃系（常時属性付与） |

> 「御札」はアクセ専用語彙。武器の「式符（W6_shikifu）」とは別物（命名競合は解消済み・Systems §4 注記）。

### 3.3 ユニーク固定配置 vs 汎用ドロップ

| 供給形態 | 対象 | 内容 |
|---|---|---|
| **ユニーク固定配置** | 全48種（mag_001〜048・v1.1） | 核となる主効果を持つ個体。隠し祠・危険地帯・ボス報酬・スニーク先取り位置に1個だけ存在する |
| **汎用ドロップ（製品版フェーズ1以降）** | 地方特性付き汎用勾玉 | 地方に応じた主効果 + 副特性1ロール。同名でも副特性の差で厳選対象になる |

**副特性ロール枠（汎用勾玉）**:

| 副特性枠 | ロール例 | 原則 |
|---|---|---|
| 攻撃系 | 攻撃 +2〜6% / 特定アーキタイプ特効 +5〜10% | 加算・独立のみ（乗算なし） |
| 防御系 | 物理防御 +1〜4 / 特定状態異常蓄積 -10〜30% | — |
| 探索系 | 銭/素材ドロップ +5〜15% | 感知系は汎用ロール対象外（鍵化防止） |
| 機動系 | 移動速度 +3〜8% / ステップ無敵 +1F | — |

### 3.4 3スロット運用と相乗・制約

**相乗のルール**: 勾玉の主効果はすべて「加算・独立」を基本とし、乗算シナジーは個別設計の勾玉のみが持つ（ACCESSORY_LIST §0.1-3 準拠）。3枠同時装備では効果が単純に合算される。

**制約のルール**: 同一勾玉を3枠すべてに装備することは可能だが、同名効果の重複は「1個分のみ有効」とする（スタック上限1）。これにより「同じ勾玉を3個集める」より「異なる役割の勾玉を3個揃える」ことが最適になる。

**相乗を意識したビルドの例**: §5 に記載。

### 3.5 体験版での勾玉配置

| Phase | 固定配置数 | 内容 |
|---|---|---|
| Phase 1A | 3個 | mag_001（杜の勾玉・ゲージ蓄積UP）／mag_013（厄除けの御守り・防御）／mag_002 配置版（スニーク先取り: 格上敵が守る祠に潜入して取る） |
| Phase 1B | 5〜8個 | 上記＋攻撃系・祓い波変化系を1〜2個追加。「どれを3枠に入れるか」を選ばせる |

## 4. 衣

### 4.1 設計方針

- **1枠・見た目変化なし・固定ラインナップ**（Systems §3）
- 防御値の素の加算 ＋ **環境特性**（地域ギミックへの適応）
- 環境特性はソフト設計: 「あると快適・なくても通れる」。特定の衣がないと進めない場所を作らない（鍵化禁止）
- スプライト変化なし（グラフィック負荷ゼロ）

## 5. ビルド例（勾玉3枠＋衣＋武器の組み合わせ）

### ビルド例A — 攻撃寄り（祓い刀軸）

| スロット | 装備 | 狙い |
|---|---|---|
| 武器 | W1_haraitou_g7 雪冴祓刀 | 連撃4段化で手数増・縛り祓いとゲージ加算の相乗 |
| 衣 | koromo_06 狩衣 | 状態異常蓄積軽減で戦闘中のデバフを抑える |
| 勾玉1 | mag_002 岩鬼の勾玉 | 攻撃+8% / 大型敵からの被ダメ -10% |
| 勾玉2 | mag_004 炎の勾玉 | 【祓い波】浄化大技を炎属性化（縛り中の敵に炎が広がる） |
| 勾玉3 | mag_034 見切りの組紐 | ステップ無敵 +2F（回避で連撃の起点を作る） |

> **厳選の楽しさ**: mag_002 と mag_004 は固定配置で入手済みとして、3枠目に攻撃系か移動系を選ぶかが個人のプレイスタイルを反映する。

### ビルド例B — 探索寄り（全地域収集）

| スロット | 装備 | 狙い |
|---|---|---|
| 武器 | W8_shakujo_g8 幽鳴錫杖 | 地霊の輪の感知範囲拡大で隠し要素を感知しやすくする |
| 衣 | koromo_10 常世の白衣 | 常世の隠し勾玉の共鳴音（探索系の環境特性） |
| 勾玉1 | mag_024 神楽鈴（かぐらすず） | 隠し祠を近接時に微震動で感知 |
| 勾玉2 | mag_026 埋火の鈴 | 埋蔵勾玉/宝の近接感知 |
| 勾玉3 | mag_029 福禄の根付 | 銭ドロップ +25%（強化資金の確保） |

> **厳選の楽しさ**: 探索寄りビルドは戦闘力が低いため、高Lv地域での危険度が上がる。「探索を快適にするか戦闘を強化するか」のトレードオフが勾玉選択の判断軸になる。

### ビルド例C — スニーク寄り（先取り・回避特化）

| スロット | 装備 | 狙い |
|---|---|---|
| 武器 | W7_maiougi_g5 宮舞扇 | 移動撃ちで逃げながら削る・ヒット数に応じ祓いゲージ加算 |
| 衣 | koromo_06 狩衣 | 状態異常蓄積 -15% でスニーク中のデバフを抑える |
| 勾玉1 | mag_033 早足の組紐 | 移動速度 +10%（スニーク移動の快適性） |
| 勾玉2 | mag_034 見切りの組紐 | ステップ無敵 +2F / 回避後踏み込み距離 +10% |
| 勾玉3 | mag_025 千里の鈴 | 画面外の格上敵の方向を予告（危険察知） |

> **厳選の楽しさ**: 組紐2枠は同ニッチに近いが、移動速度（033）と無敵フレーム（034）は加算・独立で両方有効。逃げ性能を極振りした結果、祓い波をどの属性変化に使うかが次の差別化ポイントになる。

### ビルド例D — 祓い波変化重ね（勾玉3枠を祓い波変化系に統一）

| スロット | 装備 | 狙い |
|---|---|---|
| 武器 | W1_haraitou_g8 幽閑祓刀 | 縛り祓いでゲージ即時1段加算（祓い波回しを加速） |
| 衣 | koromo_02 巫女衣 | 祓いゲージ蓄積 +5%（少しでも早く撃つ） |
| 勾玉1 | mag_009 広がりの勾玉 | 【祓い波】浄化範囲 +40% |
| 勾玉2 | mag_010 早結びの勾玉 | 【祓い波】溜め 18F→12F（リスク減・威力 -15% のトレード） |
| 勾玉3 | mag_005 水鏡の勾玉 | 【祓い波】発動後にHP小回復 |

> **厳選の楽しさ**: 祓い波変化系を3枠重ねると「頻繁に撃てる・範囲が広い・撃つたびに回復する」の複合効果で戦闘のリズムが大きく変わる。「次に勾玉を引いたらどれと入れ替えるか」が常に意識される状態を作る。
