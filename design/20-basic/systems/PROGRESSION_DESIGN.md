## 0. 役割と責務

### 本書が所有するもの

本書は「攻略とゲートの順序」を所有する。具体的には以下の4つである。

1. R1〜R10への進行順と、各地域への入域条件（ソフトゲート / ハードゲート）
2. Lv帯バンドと自由度の両立方針
3. 移動手段（徒歩→船→外洋船→飛空艇）の解放トリガーと対応地域
4. 武器 W1〜W8 の入手地域確定マッピング

### 本書が所有しないもの

- **SCENARIO_DESIGN**（story.md / SCENARIO_DESIGN.md 所有）: 物語の順序・NPCセリフ・イベントの台本
- **COMBAT_DESIGN**: 各ボスのフェーズ・HP・フレームデータ・難度調整
- **MAP_LIST**: マップ詳細・warp接続・タイル配置
- **REGION_LIST**: 地域メタ（スラッグ・色彩・マップ配分）

> 境界の原則: 「ミトがどの順番でどの地域に行けるか」は本書。「なぜ行くか（物語の動機）」は SCENARIO_DESIGN。

## 1. 攻略順とゲート設計

### 1.1 進行の基本線（R1→R10）

```
R1 始まりの里山郷 (徒歩, Lv1-8)
   └─[ハードゲート: 港解放 + 船]→ R2 瑠璃の多島海 (Lv8-16)
        └─[ハードゲート: 河港到達 + ストーリーフラグ]→ R3 宮乃京【中央ハブ】(Lv14-24)
             ├─[ソフトゲート: 推奨Lv22 + 峠越えルート開通]→ R4 霊峰の裾野 (Lv22-32)
             │     └─[ソフトゲート: 推奨Lv30 + 峠/街道（直通）]→ R5 北雪郷 (Lv30-40)
             │           └─[ハードゲート: 杉谷の隘路フラグ]→ R6 黒洲 (Lv36-46)
             │                 └─[ハードゲート: boss_11撃破で火の道開通 + 外洋の鍵]→ R7 火群と白磐の地 (Lv42-50)
             │                       └─[ハードゲート: boss_18撃破＝正しい火の証 + 外洋船（火群の湊→常世航路）]→ R8 常世（聖域・無戦闘）(Lv46-54)
             │                             └─[外洋へ + boss_13撃破＝飛空艇の鍵]→ R9 綿津見の大海 (Lv48-56)
             │                                   └─[ハードゲート: 飛空艇]→ R10 高天原 (Lv52-62)
             └─（R3 ハブから各地へ徒歩/船で自由再訪。PLAN §2-4「移動の連続性」）
```

### 1.2 ソフトゲートとハードゲートの使い分け

| 境界 | ゲート種別 | 解放条件の内容 | プレイヤーへの提示方法 |
|---|---|---|---|

### 2.2 レベルシンク無し・自由度との両立方針

- **レベルシンクは実装しない**（PLAN §2「攻略順自由」に準拠）。高Lv帯への強引な先入りは難度急上昇で自然に制止。
- **スニーク先取りは設計内**（PLAN §2-6）: R2の一部を Lv 5〜6 で突破しにいくプレイを歓迎。希少報酬はあるが必須武器は入れない。
- **追い越し対策**: 過去地域の「レア勾玉」「隠し祠の依代の恵み」は武器/移動手段解放後の再訪報酬にする（§6 再訪設計参照）。Lv 上限 Lv60 のシーリングを設けることで、適切なペースで進んだプレイヤーが R10 でちょうど飽和する。

## 6. 再訪設計

### 6.1 メトロイドヴァニア的再訪の基本方針

「後で来よう」という記憶を作り、新武器/移動手段の入手後に意味のある再訪体験を提供する。

- **武器ギミックは鍵でなくご褒美**（Systems §2 原則）: メインルートは武器ギミック非依存。ギミックの先には必ず報酬（レア勾玉・衣・手控帳ページ・隠し祠）がある。
- **視覚記号が「後で来よう」の手がかり**（Systems §2-3）: 各武器に1種の視覚記号。「白ヒビ岩 = 木槌で砕ける」を見た瞬間に思い出せる設計。
