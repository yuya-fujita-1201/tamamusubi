## 0. 役割と責務

### 0.1 本書が担保すること

本書は **消費アイテム・収集物・素材・雑貨の100点**（itm_001〜itm_100）の運用仕様を定義する。

| 責務 | 内容 |
|---|---|
| アイテム区分9種の運用ルール | 各区分の役割・使用タイミング・所持上限 |
| ホットスロット2枠の仕様 | 即時使用の操作・上限・UI方針 |
| 回復/お清め/復活の挙動 | HP・MP・状態異常回復の正確な定義 |
| 状態異常8系統×回復アイテム対応表 | COMBAT_DESIGN §6 の8系統と完全整合 |
| バフ/探索/照明の運用 | 効果時間方針・鍵化禁止の担保 |
| 収集物とイベントキーの扱い | 手控え帳ページ・物語進行品の制約 |
| 地域特産の経済接続 | 9地域1対1（v1.1）・銭/クラフト/武器強化の三重役割 |
| クラフトシステム（フェーズ1） | 素材14点・レシピ方針・解放条件 |
| 通貨=銭・ドロップ率方針 | グラインド不要バランスの根拠 |

### 0.2 EQUIP_SYSTEM との境界（重要）

| システム | 扱うもの |
|---|---|
| **本書（ITEM_SYSTEM）** | 消費アイテム・収集物・素材・雑貨の挙動・所持上限・クラフト |
| **EQUIP_SYSTEM（別書）** | 装備スロット（武器×1 / 衣×1 / 勾玉×3）の仕様・勾玉効果の詳細 |

「装備する」アイテム（武器・衣・勾玉）は EQUIP_SYSTEM の管轄。本書は「使う・集める・素材にする」アイテムのみを扱う。

### 0.3 全体の不変原則（PLAN §4 / Systems §2 をアイテムに適用）

1. **グラインド前提の設計禁止**: 回復アイテムを大量消費させるボス・バランスにしない。ホットスロット2枠・所持上限は「欲しい時に手が届く」範囲に収める。
2. **照明・鍵は進行の鍵にしない**: 提灯（itm_041）・縄・鍵類はすべて任意ルートの快適化のみ。本筋進行に必須化しない（PLAN §4 鍵化禁止）。
3. **数値インフレで区分を水増ししない**: 等級展開（HP小→中→大）は最小限。差別化は"変な効果"（FF11式）で作る。

### 1.2 各区分の運用ルール補足

**回復（HP/MP）itm_001〜012**
- ホットスロット2枠に任意でセットして戦闘中に即時使用する。
- HP回復3段階（小：おむすび itm_001 / 大：重ね握り itm_003 / 全快：椿餅 itm_004）のほか、変な効果枠（歩行継続回復・連食ボーナス・HP上げ底）で差別化する。
- MP回復は甘酒（itm_006）が序盤の基礎。神饌酒（itm_008）は社での入手限定で希少。

**お清め itm_013〜024**
- 状態異常の治療・予防・投擲の三機能に分かれる。詳細は §3 対応表参照。
- 清めの塩（itm_013）は周囲範囲の穢れ解除で世界観と実用が一致する代表品。
- 炒り豆（itm_014）は投擲消費型で鬼系に特効×2。攻撃アイテムとしても機能する。

**復活・蘇生 itm_025〜028**
- 高価・希少。起き上がりだるま（itm_025）は自動、みたま返しの札（itm_027）は手動発動で操作の差別化。
- 黄泉戸喫封じの米（itm_028）は R8/R10 限定の終盤品でテーマ性重視。

**バフ itm_029〜040**
- 戦闘前の準備として使用。同種バフは重複不可（上書き）、時間制限はすべて「戦闘1〜2回分」を目安に設定する。
- 変な効果3点（金運の小判 itm_037・慧眼の香 itm_038・倍返しの護符 itm_039）は希少枠として入手機会を限定する。

**探索・照明・鍵 itm_041〜050**
- すべて任意ルートの快適化のみ。本筋ルートに必須なアイテムをこの区分に含めない（§4 で詳述）。

**収集 itm_051〜062**
- 手控え帳ページ（itm_051〜058）は8地域1対1。完全収集で勾玉枠+1等の恒久報酬。
- 妖の図譜（itm_059）・名所の拓本帖（itm_060）・釣り手帖（itm_061）は自動的に埋まるログ型。
- 結びの口伝書（itm_062）は真エンドの条件の一部。

**地域特産 itm_063〜078**（詳細は §6）

**クラフト素材 itm_079〜092**（詳細は §7）

**イベント専用キー itm_093〜100**（詳細は §5）

### 2.3 整理UI方針

- **自動整頓**: メニュー「袋の整頓」でカテゴリ→ID順にソート。
- **捨てる**: 地域特産・クラフト素材・バフ・回復品は捨てられる。
- **捨てられない**: 収集物（itm_051〜062）・イベントキー（itm_093〜100）は捨て操作を封印。
- **所持上限到達時**: 購入・拾いに「上限に達しています」と表示し、取得をスキップ（欄外に落ちない）。

## 4. バフ/探索/照明

### 4.1 バフ系（itm_029〜040）の効果時間方針

- **基本方針**: 「戦闘1〜2回分」の短い時間制限。数値インフレ回避のため効果は+15〜20%を上限とする。
- **同種バフの重複不可**: 力の丸薬（itm_029）と力の丸薬を2個使っても効果は重複しない（新しい方で上書き）。別種バフ（力＋守り等）は共存可。
- **効果時間の目安**:

| カテゴリ | 効果時間 | 対象品 |
|---|---|---|
| 攻/守/速（基礎バフ） | 60〜90秒 | itm_029〜031 |
| 祓い波・会心・特効 | 60秒 | itm_032〜035 |
| 機動・スニーク補助 | 45秒 | itm_036・itm_040 |
| 変な効果（稼ぎ・反射・相性可視化） | 60〜120秒 | itm_037〜039 |

### 4.2 探索系（itm_041〜050）と鍵化禁止の実装方針

### 4.3 提灯=暗所の鍵化防止（Systems §5 の徹底）

- 体験版（Phase 1A/1B）の時点で、里の店（tama_musubi_sato 問屋）から提灯（itm_041）を購入可能にする。
- 御鏡（W3）の光機能と提灯は**代替手段の関係**。どちらかがあれば暗所ダンジョンを進める。
- 根の洞（ne_no_hora_b1〜boss）のマップ実装 DoD に「提灯所持で全区間を通過できること」を明記する。

### 6.3 入手経路と特産の攻略順依存性

- 地域特産は「その地域を解放してから入手できる」仕様。攻略順（R1→R10）と入手タイミングが1対1で対応する。

## 8. 通貨=銭・ドロップ率方針

### 8.1 銭の入手シンク

| 入手元 | 目安 |
|---|---|
| 敵ドロップ（dropGem） | 各敵の基底値×Lv帯補正 |
| 地域特産の売却 | 売却銭＝「その地域の武器強化1段分の1〜2割」を目安 |
| 宝箱・発見報酬 | 探索の即時ご褒美 |
| NPC 依頼報酬 | 依頼完了時の一括払い |

| 消費元 | 目安 |
|---|---|
| 武器強化（鍛冶屋） | 段階ごとに銭＋特産が必要（GROWTH_DESIGN と協調） |
| アイテム購入（店） | 回復食・バフ・探索品の補充 |
| 宿（HP/MP全快） | ワンコイン程度（チェックポイント代替） |

### 8.2 グラインド不要のバランス方針

- **1地域クリアで得られる銭**: 敵全滅+特産売却で「その地域の武器を2〜3段強化できる」量が目標値。
- **回復品の入手**: ボス部屋手前には必ず補給ポイント（宿・湧水・宝箱）を配置し、「回復が尽きたからグラインド」にならない設計。
- **ドロップ率の設計原則**:

| 素材分類 | ドロップ率目安 | 方針 |
|---|---|---|
| 汎用素材（木の枝・薬草等） | 40〜60% | 少し戦えば揃う |
| 中級素材（鬼の角・妖の羽等） | 25〜40% | 数回倒せば1個 |
| 地域特産 | フィールド採取・ボス後に必ず数個 | グラインド不要を担保 |
| 心/銭（dropHeart/dropGem） | 70〜100% | 戦闘の基本リターンを保証 |
