## 1. 地域コンセプト

始まりの里山郷は、玉結び（TAMAMUSUBI）の物語が幕を開ける「日本の原風景」そのものである。棚田・鎮守の杜・霧のかかる集落・蛍が舞う小川——プレイヤーはここで初めて祓い波を宿し、穢れを「討つのではなく祓う」ことを学ぶ。聖剣2/3 でいう「開幕・操作習得ゾーン」に相当し、移動・攻撃・回避・武器切替の基礎を無理なく習得させ、W1祓い刀→W2木槌→W3御鏡という3本の武器を順に手にしながら Lv1 から Lv8 まで自然に育つ設計である。

**物語の幕開け（序章）は霧立の里（kiritate）で起こる。** 夕暮れ、ミトが故郷・霧立の里の鳥居前で石段を掃いていると、空から傷ついた式鳥**シロ**が落ちてくる。シロは七年前に消えた父・**廻人（めぐりと）** の声で「勾玉を、集めてくれ」と告げ、ミトの掌に**結びの勾玉（itm_093・片割れ）** を残して光に還る。勾玉に触れた瞬間、ミトの体に祓い波が宿る。直後、里の**大鳥居（依代）** の光が弱まり裏山から穢れた精霊が現れる——これがチュートリアル戦闘であり、育ての親である老巫女・**鈴代（すずよ）** が駆けつけ、去り際に本作を象徴する一言を残す：「**討つんじゃないよ。祓うんだ。**」。里の門が開き、遠い水平線の異様な明るさ（常世の光）と雲間の白い影（高天原）を背に、ミトは父を追って旅立つ。

**第一章の核は「鎮守の森の喪失と回復」。** 里の鎮守の森の光が弱く、奥の**根の洞（ne_no_hora）** には穢れが満ちている。父・廻人が遺した**手控え帳の最初のページ**が里山の奥に残されており、ミトはそれを頼りに森へ分け入る。最深部で待つ**岩鬼の王（boss_01）** は、かつて里を守った山の神が、帰れない魂たちを抱え込んで穢れた姿である。祓われた瞬間、岩鬼の王は「怒り」ではなく**「安らいだ輪郭」** を見せて還っていく——世界観を象徴する最初の光景だ。鎮守の森の鳥居が輝きを取り戻し、ミトは父の**手控え帳**と、海へ出るための**港**を手に入れる。

地域の骨格は「**故郷（霧立）→ 里の中心（四つ辻）→ 杜／棚田 → 根の洞**」という奥行き構造。GLOSSARY の固有名詞「穢れ（けがれ）」「浄化」「祓い波」「勾玉（まがたま）」「手控帳（てびかえちょう）」はすべてR1で初出・教示する。鈴代の「祓うんだ」も、根の洞の壁絵（魂が天へ昇る浮彫）も、ここでは意味が説明されない——終幕で回収される「種」として配置する。

## 2. 色彩・画面設計の方針

ART_BIBLE の「25%縮小でも構造が読める前提」を里山の地形要素で実現する。

- **主色**: 新緑（若草色・黄緑系）。野外マップの草地・茂みの基底パレット。
- **対比**: 棚田の黄金（秋の稲穂色）。tanada・kawagishi などの水田地形で差し込む。季節感の演出軸。
- **差し色**: 杜の深緑（鎮守の森の暗緑）。chinju_sando・morioku でコントラストを付け「聖域感」を出す。
- **光源**: 序章の霧立の里（kiritate）は**夕暮れの暖橙**（石段・鳥居をシルエットで立てる物語の起点光）。朝〜昼の探索域は柔らかい南からの自然光。hotaru_ogawa（夕暮れ）は暖橙のグラデーション。根の洞ダンジョン群は苔光の緑＋松明の橙で狭い対比を作る。
- **縮小時可読性**: 石垣・田の段差・鳥居のシルエットで地形種別を即認させる。同系色の野外マップが連続するため、接続先ごとに1アクセントカラー（霧の青／稲穂の黄／杜の深緑）を割り当てて区別する。

## 4. 導線設計

> 接続の正典は [MAP_TRANSITION_GRAPH §3 R1](../../10-lists/MAP_TRANSITION_GRAPH.md)。以下はその無向グラフ上の歩行ルート設計。

- **物語スポーン**: プレイヤーは **kiritate 霧立の里** に出現する。序章イベント（シロ・鈴代・最初の祓い）を経て里の門が開き、**satoyama（里の中心＝四つ辻）** へ抜ける。kiritate ↔ satoyama ↔ aze_michi の三叉が R1 西側の入口となる。
- **メインルート（背骨貫通）**: kiritate（序章）→ satoyama（四つ辻ハブ）→ takadai → tanada → hotaru_ogawa → ne_no_hora_ent → ne_no_hora_b1 → ne_no_hora_b2 → ne_no_hora_mirror（W3御鏡入手）→ ne_no_hora_boss（boss_01 撃破）→ 港解放。故郷から根の洞最深部・港まで途切れず繋がることを「背骨貫通」原則とする。なお根の洞入口へは hotaru_ogawa 経由（棚田ルート）と higashizaka 経由（東坂ルート）の**二系統**で到達でき、探索の自由度を担保する。
- **杜・聖域ルート**: satoyama →（aze_michi → kuririn →）chinju_sando → morioku（mag_001 勾玉）／ chinju_yashiro（手控帳ページ）。鎮守の社（chinju_yashiro）と拝殿（sato_jinja_haiden）は祈祷でHP全回復できる聖域として機能し、地下祠（chinju_yashiro_ue）への潜入で衣 koromo_02 を先取りできる。
- **里の回廊（拠点）**: satoyama ↔ tama_musubi_sato を入口に sato_kajiya（武器強化）・sato_yado（HP回復）・sato_donya（特産換金）・sato_yakuba（依頼受注／里長）・sato_jinja_haiden（祈祷回復）をいつでも往復できる。ボス前のセーフゾーン兼補給拠点。
- **寄り道（収集）**: takadai → uemura_hatake → kofun_iwa（勾玉 mag_003）の北西ループ、kuririn → tanuki_hokora（隠し商人・特産）、hotaru_ogawa → kakushi_izumi（隠し回復祠）。いずれも本筋ゲートを塞がない自由探索。
- **スニーク/先取り**: 高Lv強敵の岩鬼の眷属（enm_008）が kuririn・ne_no_hora_ent に配置されるが、W1祓い刀の茂み斬りで外周を回り chinju_yashiro_ue に潜れば衣 koromo_02 を先取り可能。ボス手前の ne_no_hora_b2 では W2木槌の「もろい岩砕き」でショートカット穴を掘り第一層を飛ばせる（PLAN スニーク設計）。

## 5. 敵構成・難易度カーブ

R1 は Lv1〜8 の操作習得ゾーン。ENEMY_LIST §1.1（enm_001〜008 の8種）が配置される。世界観どおり、敵は「討つ」のではなく祓い波で「**祓う**」——浄化されて安らかに還る演出を全戦闘の基底に置く。

**Lv1〜3（霧立・里の中心周辺）**:
- enm_001 山鼠（swarm）: 最弱。群れで寄るが1撃で祓える。操作の初回教示。kawagishi/tanada/ne_no_hora_b1 では泥団子を投げる遠隔変種として再登場。
- enm_002 野犬の群れ（swarm/lunge）: 突進テレグラフの読み方を学ぶ2番目の壁。W1祓い刀推奨。
- enm_003 木霊（ranged）: 遠隔胞子弾の回避と近接接近の基礎。眩みデバフの初出。序章のチュートリアル戦闘もこの系統。

**Lv2〜5（杜・参道・棚田）**:
- enm_004 案山子の付喪（hop）: 跳ね接近の付喪（道具の妖）系代表。W2木槌の打撃特効を初めて意識させる段。縛りデバフの初出。
- enm_005 狐火（fly）: 浮遊敵の初出。地上攻撃が届かない場面を演出し「飛行に弱い武器がある」ことを示唆。W5和弓は未所持のため距離を取るか回避で対処。R2解禁後に「あの敵が楽に倒せる」を実感させる伏線。
- enm_006 川蟹（tank）: 正面ガード導入。W2木槌の殻砕きが有効なことを里人のヒント台詞で示す。

**Lv3〜7（根の洞 入口〜第二層）**:
- enm_007 里蝙蝠（fly/ambush）: 洞窟天井からの急降下奇襲。暗所での警戒を強いる。
- enm_008 岩鬼の眷属（lunge）: 鬼系最強雑魚。炒り豆 itm_014 の特効を覚えていれば楽だが知らなければゴリ押しになる。ダンジョン外周の kuririn でも出現し「早めに覚えれば楽」の誘導を作る。

**弱点・属性の配置方針**: W1祓い刀（全体基礎）→W2木槌（殻/打撃系）→W3御鏡（浄化/光反射）の順に実戦で使う機会を作る。祓い波属性は R1 では「木霊・狐火・岩鬼系」の浄化光が基礎。火点き付与（狐火）とお清め（鎮め草 itm_017）の関係を R1 内で教示する。

## 7. 移動解放・進行ゲート

R1 の進行ゲートは **boss_01（岩鬼の王）の撃破** 一点。

- **撃破直後**: 鎮守の森の鳥居が輝きを取り戻す演出。父・廻人の**手控え帳（全ページ）** が手に入り、地図には列島各地の依代・海の彼方（薄い金色のにじみ＝常世）・雲の上（霞んだ白い印＝高天原）が描かれている。里長から**里長の書状（itm_094）** が渡される。
- **港の解放**: R1 北縁の港マップ（kakouko・R1/R2 連絡マップ扱い）が開通し、船に乗れるようになる。これが R2「瑠璃の多島海」への唯一の移動手段（[REGION_LIST §1.3](../../10-lists/REGION_LIST.md)・[PROGRESSION_DESIGN](../systems/PROGRESSION_DESIGN.md) 参照）。
- **武器ゲート**: W1祓い刀（開始時所持）→W2木槌（takadai 付近で入手）→W3御鏡（ne_no_hora_mirror で入手）の順。W3 入手前はボス部屋への突入が困難（御鏡なしだと第2相の封印柱を怯ませられない）。
- **探索ゲートは最小化**: chinju_yashiro_ue への潜入は boss_01 攻略前でも可能（衣 koromo_02 の先取り）。移動解放の本筋ゲートは boss_01 のみに絞り、探索の自由度を損なわない（FF6/FF9 方式の連続性）。
