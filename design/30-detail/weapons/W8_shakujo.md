## 1. 戦闘ニッチ・役割

錫杖は玉結び8系統のうち唯一「三流派スキルを増幅する」支援武器である。装備中はすべての三流派アクティブスキル（結び skl_001〜008・神楽 skl_009〜016・式 skl_017〜024）に以下の補正が常時適用される（[COMBAT_DESIGN §8.2](../../20-basic/systems/COMBAT_DESIGN.md)）。

- **スキル消費 -30%**: 祓いゲージ / MP 消費量を 30% 軽減（→ g9 で -42% が上限値）
- **スキル効果 +20%**: ダメージ・効果量・持続時間を 20% 増加（→ g9 で +30% が上限値）

この増幅効果は武器を持ち替えた瞬間に切り替わる「装備中パッシブ」であり、スキル非使用プレイヤーを疎外しない設計として「突き鳴らし（周囲 AoE 多段）」という固有の通常攻撃を備える。スキルビルドを組まないプレイヤーにも手応えを担保し、死に武器化を防ぐ。

他7系統との差別化：

| 軸 | 錫杖 | 他系統 |
|---|---|---|
| 単体火力 | 最低（g1: 0.80） | 和弓 g1: 2.20、木槌 g1: 1.80 |
| 支援対象 | 三流派スキル全種 | 自己完結型 |
| スキルシナジー | 最大（消費軽減＋効果増幅） | 無し〜軽微 |
| フィールド用途 | 地霊の輪（隠し要素感知） | 地形特化（岩破壊・光パズル等） |

終盤（フェーズ4・R10 高天原）入手のため、スキルを取り揃えたプレイヤーに最大の恩恵が出る遅咲き武器でもある。強化が進むと突き鳴らし自体の AoE も拡張し、「チャージ技 響鳴 + スキル連発」の二択戦術が成立する。

## 3. チャージ技の段階進化

### 3.1 チャージ技「響鳴（きょうめい）」

錫杖の固有チャージ技。長押し入力でリングの音が響き渡り、一定時間「祓いゲージの蓄積速度」を増幅する。

| 進化帯 | 段階 | 響鳴の性能 |
|---|---|---|
| 基本形 | g1〜g3 | ゲージ蓄積速度 **×2.0**（3秒間）。自身のゲージにのみ作用。発動中は淡い金のリング演出 |
| 強化形 | g4〜g7 | g7 到達で蓄積倍率が **×2.5** に上昇（WEAPON_LIST §1.8 g7 「響鳴中の蓄積を×2.5へ」と同値）。持続は引き続き3秒。スキル連発との相性が大きく高まる段階 |
| 最終形 | g8〜g10 | g10「天鳴錫杖」で **パーティ全体のゲージに加算**が解放される（WEAPON_LIST §1.8 g10 「響鳴がパーティ全体ゲージ加算」と同値）。多人数プレイ想定ではないが、将来の協力拡張時にも設計が生きる |

- 響鳴の発動中は通常攻撃（突き鳴らし）が引き続き可能。スキル即打ちも阻害しない
- キャンセルタイミング: 響鳴中に被弾してもゲージ倍率は継続（途中で中断しない設計）
- 祓いゲージが既に満タンの場合、蓄積速度増幅は無効になるが倍率数値は保持される

### 3.2 通常攻撃「突き鳴らし（つきならし）」の段階深化

突き鳴らしは地面を突いて周囲に衝撃波 AoE を発生させる多段攻撃（近接・周囲リング状）。

| 進化帯 | 段階 | 突き鳴らしの性能 |
|---|---|---|
| 基本形 | g1〜g2 | 周囲 AoE 2段ヒット。倍率 0.80〜0.83。ノックバックなし |
| 中盤 | g3〜g5 | g3 で AoE 段数 +1（3段化）、g5 で小ノックバック付与（押し返しによるスペース確保） |
| 後半〜最終 | g6〜g10 | 倍率が 0.95〜1.08 と攻撃武器の g1 水準近傍まで成長。g10 で判定が **全周大 AoE** に拡張し、群体への圧力が格段に増す |

フレームデータ方針（[COMBAT_DESIGN §3.1](../../20-basic/systems/COMBAT_DESIGN.md)）：かまえ 3F ／ 判定 4F（周囲）／ 硬直（HS）4F ／ 後隙 8F。支援武器としては平均的な後隙で、スキルへの繋ぎがスムーズになる設計。

## 4. フレームデータ方針

> 実数は [COMBAT_DESIGN §3.1](../../20-basic/systems/COMBAT_DESIGN.md) で確定。ここでは方針のみ記述。

- **かまえ（3F）**: 他近接武器より短めに抑え「置いておくだけで機能する支援武器」の印象を与える
- **判定（4F・周囲 AoE）**: 前方集中型ではなくリング状。立ち位置を選ばず囲まれても使える
- **後隙（8F）**: 木槌（12F）より軽く、刀（6F）より重い。突き鳴らし後に即スキル打鍵ができる程度の間合い
- **スキルとのフレーム接続**: 突き鳴らし後隙中にスキルコマンドを受け付け、後隙終了後にスキルを発動するバッファリングを推奨設計とする（スキル誘発の失敗ストレスを排除）
- **響鳴の発動フレーム**: チャージ溜め開始から 60F（約 1.0 秒）で最低限の蓄積効果。長押し継続で効果時間を消費。最大効果は溜め 120F 到達時

## 5. フィールドギミック（探索用途）

### 5.1 地霊の輪（ちれいのわ）

錫杖を構えた状態で地面を軽く突くと、足元から薄い光の輪が広がる（地霊の輪）。近くに**隠し要素**（埋蔵勾玉・隠し祠・封印された古道の入口など）がある場合、輪がわずかに揺らいで色が変わる。

**絶対禁止事項（アイテム鍵化禁止の原則）**: 地霊の輪が反応する対象は「隠し要素」に限定し、**必須の進行経路・ストーリートリガー・ボス到達の鍵には絶対に使わない**。地霊の輪を持たないプレイヤーが詰まる状況を作ってはならない（同原則は skl_013「峰からの見晴らし」にも適用される・[SKILL_LIST §1.2](../../10-lists/SKILL_LIST.md)）。

| 用途 | 詳細 |
|---|---|
| 隠し勾玉の感知 | 地中・岩陰・木の洞に埋まった勾玉に反応。掘り出しには「結びの手」が必要な場合もある |
| 隠し祠の感知 | 草木に覆われた小祠。反応したら周囲を調べると祠が現れる（勾玉・古文書など） |
| 古道の入口感知 | 解放前の隠し通路。必須の場所には絶対配置しない |
| 感知なし対象 | 宝箱（外から見える）・NPCイベント・ボス扉・任意の橋や鍵——これらは鍵化禁止のため対象外 |

g8「幽鳴錫杖」で感知範囲が拡大し、より遠くの隠し要素に反応するようになる。ただしゲームプレイ上の必須性はゼロのまま維持する。

### 5.2 鐶音（かんね）の演出

錫杖の鐶（金属の輪）が揺れる音は段階が上がるほど澄んで遠くまで届く演出に変化する。音は戦闘中の突き鳴らしSEにも影響し、g1 は鈍い打音、g10「天鳴錫杖」は広い金属響鳴に変わる。サウンドデザイン側への発注仕様として本記述を残す。

## 6. 入手・強化の地域動線

### 6.1 g1 入手（R10 高天原）

- R10「高天原（takamagahara）」クリア後、または高天原の神鍛冶「天鍛場」で購入
- 入手時点でプレイヤーはほぼすべての三流派スキルを習得済みの想定（24スキル中の大半）
- 入手即座に消費軽減と効果増幅が全スキルに乗るため、終盤ボスへの再挑戦・取りこぼし探索に直結する動機を持つ

### 6.2 g2〜g9 強化（R10 再訪 + 全地域再訪）

g2〜g9 はいずれも R10 再訪（入手後の鍛冶屋アクセス）で強化可能。特産は全地域から集める設計になっており、「R10 到達後に全地域を再巡する」動機になる。

| 段階 | 必要特産 | 調達地域 |
|---|---|---|
| g2 | 山鉄（itm_063） | R1 satoyama |
| g3 | 漆（itm_068） | R3 miyanokyo |
| g4 | 潮塩（itm_066） | R2 tatoukai |
| g5 | 金箔（itm_067） | R3 miyanokyo |
| g6 | 火打石（itm_069） | R4 reihou |
| g7 | 氷晶（itm_071） | R5 hokusetsu |
| g8 | 夜光苔（itm_074） | R6 kurosu |
| g9 | 常世貝（itm_075） | R8 tokoyo |
| g10 | 神鉄（itm_077） | R10 takamagahara（希少） |

### 6.3 g10「天鳴錫杖」（R10 希少素材）

神鉄（itm_077）は R10「高天原」の最深部か、高天原クリア後の隠し採掘場でのみ入手できる希少特産。ゲームクリア後・全スキル取得後に錫杖を育て切るプレイヤーへのご褒美段階と位置づける。
