## 1. 戦闘ニッチ・役割

舞扇は近接8系統のうち唯一「移動しながら振れる」武器であり、機動戦術の中核を担う。

**差別化の核（他7系統と直交する3点）:**

1. **移動撃ち専用ニッチ**: 近接4種（祓い刀・木槌・薙刀・舞扇）の中で唯一、攻撃モーション中に移動入力が有効。かまえ2F・後隙3Fという最短クラスのフレームデータが「動きながら削り続ける」プレイを支える（COMBAT_DESIGN §3.1参照）。

2. **手数最多・単発最弱のトレードオフ**: per-hit 0.42（g1）〜0.65（g10）は全武器最低。しかしヒットストップ2F×hit（全武器最軽）で手数を積み重ねると総ダメージは中堅以上に到達し、g9「削り重ね（連続ヒットで被ダメ重畳）」が実質的な火力底上げになる。

3. **風属性・飛行妖特効**: 風属性で飛行妖に ×1.5（g6 火舞扇達成後は ×1.8）。`fly` 行動enumの敵（大海月・寒蝉・常世蝶など）への専門対策武器として運用できる。遠隔武器（W5和弓も飛行妖 ×1.5 を持つ）と用途が重なるが、舞扇は近接多段で祓いゲージを稼ぎながら戦える点で差別化される。

**弱点（万能化させない設計):**
- 単発ダメージが低いため、重装妖や高防御ボスには当たり負けが大きい（木槌に任せるのが最適）
- リーチは最短クラス（攻撃判定が自身周囲近傍のみ）。lunge型・tank型の敵への正面突撃は自滅リスクが高い

**推奨プレイスタイル**: runner型敵（鼠小僧 enm_018）の追跡、fly型敵との空対地戦、swarm型の高機動散らし。「当てながら逃げる」「追いかけながら削る」が最も効果を発揮する局面。

## 3. チャージ技の段階進化

チャージ技「神楽の風」は EQUIP_SYSTEM_DESIGN §2.2 の3帯構造（基本形/強化形/最終形）に従って進化する。

### 基本形（g1〜g3）: 突風による距離確保

「神楽の風」は自身中心の小範囲（半径約2マス）に突風を起こし、周囲の敵を吹き飛ばす。主目的は**距離確保**——至近距離に詰められた際の緊急離脱。ダメージは控えめ（基本倍率×0.8程度）だが、ノックバックの距離が大きく、fly型・hop型敵との間合いリセットに機能する。

- g1: 半径約2マス・前方優先ノックバック・飛行妖ヒット時 ×1.5 属性ボーナス適用
- g2: 段数増加（攻撃手数が増え、チャージ技へのゲージが溜まりやすくなる）
- g3: 風属性ダメ +10% により神楽の風のダメージも底上げ（風属性の全ヒットに適用）

### 強化形（g4〜g7）: 射程拡大と機動補完

g4 から神楽の風の判定範囲が拡大し（射程+1マス）、より広い空間で複数の fly型敵を同時に吹き飛ばせるようになる。g7「雪舞扇」では移動撃ち中の手数低下が解消されるため、チャージ技のみならず通常攻撃の手数も最大に保ちやすくなる。

- g4: 神楽の風の射程 +1マス（半径約3マス相当）。swarm型敵を散らすほどの範囲化
- g5: 通常攻撃ヒット数に応じた祓いゲージ加算。チャージ頻度が間接的に上昇
- g6: 飛行妖への倍率 ×1.8 が適用。神楽の風の吹き飛ばしも同倍率でダメージ増
- g7: 移動中手数低下解消。「逃げながら削る」プレイスタイルの完成形

### 最終形（g8〜g10）: 竜巻化（吹上＋持続）

g8 で神楽の風に「範囲鈍足付与（donsoku）」が加わり、吹き飛ばしに加えて敵の追撃速度を5秒間落とす制御技に変貌する。g10 では竜巻化——判定が上方向にも発生する「吹上」効果が加わり、fly型敵を強制着地させる（実質スタン類似）と同時に、竜巻が数秒間フィールドに留まって継続ダメージを与える。

- g8: 神楽の風が範囲鈍足付与（donsoku 5秒）。吹き飛ばし後も動けなくなることで継続ダメージの機会が生まれる
- g9: 連続ヒット時の「削り重ね」が神楽の風の吹き飛ばし前に重畳する場合、蓄積ダメージが一気に炸裂するコンボが成立
- g10: **竜巻化**——神楽の風が竜巻に変化。効果は以下の2点:
  - **吹上**: 飛行妖を強制的にダウン状態（着地）させる。地上攻撃が通りやすくなる
  - **持続**: 竜巻が発生地点に3〜4秒間留まり、近くの敵に per-tick ダメージ（風属性 ×1.8）を継続付与。この間プレイヤーは自由に動ける

> チャージ技の発生F目安は通常攻撃と同じ 2F かまえ（COMBAT_DESIGN §3.1「W7_maiougi: かまえ2F」準拠）。後隙は竜巻召喚後も移動可能なため実質 3F のまま。

## 4. フレームデータ方針

COMBAT_DESIGN §3.1 の確定方針に基づく。実数はプロトタイプで計測・確定する。

| 指標 | 舞扇の方針 | 根拠・意図 |
|---|---|---|
| かまえ（準備F） | 2F（全武器最短クラス） | 「振るたびにテンポが止まらない」操作感。移動と攻撃の切り替えを滑らか化 |
| 判定（ヒット判定F） | 2F×n hit（多段） | n は段階で増加。g1〜g2 は n=4〜5 hit、g10 では n=7〜8 hit 想定（プロトで調整） |
| HS（ヒットストップ） | 2F/hit | COMBAT_DESIGN §1.3「移動撃ちを妨げない最小HS」。重さを感じさせず手数の軽快さを演出 |
| 後隙（硬直F） | 3F | 近接4種中最短。次の移動入力までのディレイが事実上ない設計 |
| 攻撃中の移動入力 | 有効（唯一の仕様） | 攻撃モーション中に移動可。判定発生位置はキャラクター追随 |
| リーチ | 最短クラス（隣接1〜1.5マス） | 機動力とトレードオフ。dodge → 近接 → dodge のリズムを要求 |

> **移動撃ちステート**: 攻撃中の移動入力を有効にする状態機械への影響については、COMBAT_DESIGN §11.2「舞扇の移動撃ちステート：フェーズ3着手前にスパイク確認（Systems §7-4）」が確認事項として残っている。本版では方針のみ定め、実数確定はスパイク後。

## 5. フィールドギミック（探索用途）

「アイテムを鍵にしない原則」の代替として、舞扇の風属性を使ったフィールド操作が進行の選択肢を広げる。**必須経路への配置禁止**（鍵化禁止原則 SCOPE_AND_SCALE §4 準拠）。

### 止まった風車
特定の地点に設置された風車（石臼・水車・揚水装置などに連結）が錆び・重みで止まっている。神楽の風（または通常攻撃の連続ヒットの「風圧」）を当てると羽が回り始め、連結先の仕掛けが動く。

- **用途例**: 高台の揚水装置を回すと水路が通じ、先の霧だまりが晴れる。止まった製粉小屋の扉ロックが解除される
- **鍵化しない設計**: 風車ギミックのある経路には必ず迂回路が存在する。風車を使えば「近道」や「追加宝箱」が開く報酬型

### 霧だまり
常世由来の濃霧が特定のエリアを覆い、視界が大幅に低下する（暗所ほどの完全暗転ではなく、視界が 60〜70% に狭まる状態）。神楽の風を霧の中心に放つと霧が晴れ、10〜15秒程度の視界回復が得られる。

- **用途例**: 霧だまりの中に隠れた小祠・勾玉の配置発見。霧の中の飛行妖（常世蝶 enm_???）を追いやって戦闘を回避
- **g8「霧払扇」との命名連動**: g8 の段階名が「霧払扇」。霧だまりに当てた際の晴れ範囲と持続時間が g8 以降は大幅に増加（g1〜g7 は小範囲・短時間）

### 帆掛け舟の操作
海岸・川沿いの帆掛け舟が向かい風で出られない状況で、神楽の風を帆に向けて放つと舟が移動する。

- **用途例（R8 tokoyo）**: 常世の入江で帆掛け舟を使い対岸の小島へ渡る。対岸には宝箱・隠し勾玉・NPCが配置される
- **射程との関係**: g4 で神楽の風の射程が +1 マス拡大するため、足場が悪い状況での舟操作が楽になる

### 通常攻撃による軽量オブジェクトの移動
舞扇の多段攻撃（通常攻撃）は per-hit の風圧判定を持つ。軽量の仕掛け物（吊り提灯・紙風船・薄板の目隠し壁）に当てると押し流せる。

- 重いオブジェクトは動かない（木槌の仕事）。舞扇は軽量・薄板に特化したギミックに対応する

## 6. 入手・強化の地域動線

### g1 入手（R8 tokoyo）

体験版の対象外。製品版フェーズ3で R8 `tokoyo`（常世の海）に到達した際に入手可能になる。入手経路は以下の2ルートが想定される（詳細は REGION_LIST R8 基本設計で確定）。

- **ルートA（購入）**: tokoyo の港町・職人街にある扇師の工房で購入。銭のみで入手。
- **ルートB（発見）**: 入江の帆掛け舟ギミックを解いた先の小島の宝箱（推奨・チュートリアル的な探索動線として、ギミックを最初に舞扇で解くことでそのまま手に入る自己紹介設計）

### g2〜g5 強化（R1・R2・R3 再訪）

g2〜g5 の強化素材は序盤〜中盤地域の特産を流用する（山鉄 itm_063 / 漆 itm_068 / 珊瑚 itm_065 / 金箔 itm_067）。R8 入手後にバックトラックして購入するか、R8 入手前に余剰で集めた素材が流用できる。「最新武器を入手してすぐに中盤強度に上げられる」利便性を提供し、フェーズ3の快適なスタートラインを確保する。

### g6〜g8 強化（R4・R5・R6 再訪）

g6（火舞扇 / 火打石 itm_069）・g7（雪舞扇 / 白銀 itm_072）・g8（霧払扇 / 夜光苔 itm_074）はそれぞれ R4・R5・R6 の地域特産を要求する。フェーズ3の探索進行中に R4〜R6 を再訪するか、在庫があれば連続強化できる。g6 達成時点で飛行妖 ×1.8 が解放されるため、R8 ボス（fly/debuff 型の常世飛行妖想定）への対策として攻略前に g6 到達を推奨する自然な動線が生まれる。

### g9 強化（R8 ）

g9（常世舞扇 / 潮玉 itm_076）は R8 tokoyo の地域特産。R8 を十分に探索するほど素材が手に入り、フェーズ3のクライマックスに向けた強化として機能する。

### g10 強化（R10 takamagahara）

g10（天羽扇 / 天羽根 itm_078）は R10 takamagahara の希少特産。フェーズ4（最終エリア）到達後に強化可能。舞扇の最終形「天羽扇」は竜巻化チャージ技が解放され、最終ボス戦・高難度コンテンツに向けた完成形となる。

**強化スケジュール概観（銭額目安）:**
- g2: 銭 50
- g3: 銭 100
- g4: 銭 200
- g5: 銭 400
- g6: 銭 700
- g7: 銭 1000
- g8: 銭 1500
- g9: 銭 2500
- g10: 銭 5000 ＋ 天羽根（希少・R10 固定配置）

> 銭の実数値は [GROWTH_DESIGN §2](../../20-basic/systems/GROWTH_DESIGN.md) で調整・確定する。上記は WEAPON_LIST §0.3「g2:50 → g10:5000 銭程度」の目安に準拠した参考値。
