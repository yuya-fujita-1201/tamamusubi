## 1. 地形レイアウト方針

「稜線の峠道——横風が削り続けた白い砂利道と、その両脇に積み上がる雪壁」。稜線を横断する1本のS字ルートが主役であり、東西を分かつ分水嶺の頂点（H2）を目指して南（r5_yukidamari_mori 側・H0）から登り、東（r5_reika_kougen 側・H1）へ下る構造。強風ギミックが峠道全域に作用し、吹き曝しの稜線（H2 頂点付近）では風圧でキャラクターが東向きに流される。両脇の雪壁（kaIshigaki 石積み積雪版・H2 段差）が防風壁の役割を兼ね、壁の切れ目で一時的に風を遮りながら進む。白銀の稜線と、遠方に広がる藍色の山稜が対比を成す。主役=峠道を白く埋める雪壁と強風エフェクト、背景=稜線から望む藍色の山塊と吹雪の白モヤ。

## 2. タイル方針

- 峠道（H1 中段通路）: `kaYukihara`（雪原・設計値・実装時に tile_contract.json へ登録）を幅3〜4タイルで敷き、踏み固められた轍表現とする。
- 雪壁・段差（H2 両壁）: `kaIshigaki`（石積み）に積雪重ね描きを施した雪壁タイルを配し、段差前面下に `waDropshadow` frame3 を敷く。H2 上面は `kaDote`（段差）で H1 との境を明示する。
- 稜線頂点（H2 中央ピーク）: `kaKasumi`（霞・設計値・実装時に tile_contract.json へ登録）と雪原タイルを混在させ、吹雪中の彩度低下と白モヤを前景に重ねる。
- 吹雪中の表示: 視界は周期的に悪化し、彩度が低下して白モヤが前景に掛かる。敵は気配シルエットとして輪郭のみ表示。
- 色彩: 主色=純白〜青みがかった白。遠景の山稜=藍（#3A4A6A）。段差影=`waDropshadow` frame3。25%縮小でも S 字ルートの輪郭と雪壁の段差ラインが読めること。

## 3. 高低差設計（H0 / H1 / H2）

> §10.2 の heightmap 入力の設計意図をここに文章で残す。装飾の前に高さを決める。

| レベル | このマップでの意味 | 代表エリア |
|---|---|---|
| H0 | 低地: 南入口の積雪谷底・吹きだまり | マップ南端 y=6〜7 の南入口（r5_yukidamari_mori 側） |
| H1 | 中段: 峠道本体・S字ルート通路 | マップ中央縦軸 y=2〜5 を縫う歩行ルート |
| H2 | 高所: 稜線頂点・雪壁の上端・強風最大帯 | マップ北寄り中央 y=0〜1 の頂上付近・両壁の縁 |

- H0/H1 境界（南入口の段差登り）に `kaDote` 段差を配置。
- H1/H2 境界（雪壁の上縁）に `kaIshigaki` 石積みを配置。
- 擁壁前面行の真下セルに `waDropshadow` frame3 を敷く。

## 4. 歩行範囲・導線

- 主要導線: 南端 r5_yukidamari_mori ワープ（H0）→ `kaDote` を登り H1 峠道へ → S字に蛇行しながら北上 → H2 稜線頂点を越え → H1 に下り → 東端 r5_reika_kougen ワープ（H1）。南東から北西へ、さらに北東へと折れる1本道が途切れず通じる。
- 雪壁（H2 両壁）は歩行不可。壁の切れ目（2〜3箇所）のみ H2 上段へ踏み込める（隠し宝箱エリア）。
- 強風の吹き曝し帯（H2 頂点付近）は H1 道より幅が狭く、雪壁の切れ目で一時的に風を遮れる。H1 の幅広な峠道では風圧スライドが軽減される。
- 頂点付近で足を止めると東向きに流され、道幅ギリギリで踏みとどまる必要がある（強風ギミック詳細は §6）。

## 6. ギミック

- **強風（横流れ）**: H2 稜線頂点帯全域で常時東向きの風圧が作用する。プレイヤーが立ち止まると毎秒 0.5 タイル東へスライドし、峠道端の崖縁（H0 落下エリア）まで流されるとダメージ＋南端リスポーン。雪壁の切れ目（H2 壁の隙間）または H1 道の広い区画で一時停止すると風圧が解除される。W6火符を使った「雪壁溶解」で壁に穴を開けて新たな防風地点を作るサブギミックを設ける（進行必須ではなく探索的）。
- **吹雪の視界変化**: 30秒サイクルで視界が悪化（彩度低下・白モヤ前景）→回復を繰り返す。視界悪化中は enm_035 が気配シルエット化し遠距離認識が困難になる。
- 照明・鍵を進行の条件にしない原則を守る。

### 10.3 DoD 5本柱への対応メモ

[MAP_DEFINITION_OF_DONE](../../../../quality/MAP_DEFINITION_OF_DONE.md) の各柱を、このマップでどう満たすか1行ずつ：

- **構造**: H0（南入口吹きだまり）／H1（S字峠道）／H2（稜線頂点・雪壁）の3レベルが共存し、H0/H1 境界に kaDote・H1/H2 境界に kaIshigaki を置いて段差を明確化する。
- **視認性**: 25%縮小でも純白〜青みがかった白の S 字ルートと、藍色（#3A4A6A）の遠景山稜のコントラストにより峠道の輪郭が読める。
- **歩行範囲**: 南端 H0 → kaDote 登り → H1 S字道 → H2 頂点越え → H1 → 東端の導線が途切れず通じる。
- **主役と背景**: 主役＝H2 稜線の雪壁と強風エフェクト・白モヤ前景／背景＝藍色の山塊遠景と吹雪の空。
- **タイル境界**: kaYukihara（峠道）／kaIshigaki 積雪（雪壁）／kaKasumi（白モヤ）の3系統を段差で明確分割し、境界ノイズを抑える。
