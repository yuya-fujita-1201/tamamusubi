## 1. 地形レイアウト方針

主役は「嵐の海域を渡る船上の激戦」。荒波に翻弄される甲板（H1）と傾いたマスト上（H2）を舞台に、海賊頭と水竜の子が待ち構える。船体は沈没船の残骸を横目に嵐の中を進む設定で、背景には稲妻が走る暗雲・砕け散る波頭・半没した他船の船首が遠景に見える。連絡マップだが戦闘あり設計のため、アクションとして機能する高低差（帆柱スイング）を核に置き、その後に東端の水竜の子との決戦を配置する。甲板は前半（西）から後半（東）へ直進し、途中で帆柱スイングが高低差移動の選択を提示する。最東端を越えると嵐が晴れ、静かな河港（link_kako_ko）＝宮乃京の玄関が迎える演出落差が旅程の達成感を作る。

## 2. タイル方針

- 傾く甲板（H1主動線）: `shipDeckWet`（濡れた板甲板）。嵐の降雨で板目が滑る表現。継ぎ目を横方向（東西軸）に走らせて船の前後進を強調。
- 舷側・手すり（甲板縁）: `shipBulwarkDamaged`（破損舷墻）＋`shipRailing`（手すり overlay）で H1 と H0（荒れる海面）の境界を明示。一部手すりが崩れた演出箇所あり。
- マスト・帆桁（H2平場）: `shipMastBase`（帆柱基部）＋`shipYard`（帆桁 overlay）を甲板中央〜後方に1〜2本。帆柱スイングの起点。
- 沈没船残骸（背景装飾）: `shipWreckBow`（半没した船首）を H0 海面上に overlay 配置。ゲームプレイに影響しない純背景。
- 荒れる海面（H0・舷外）: `seaStorm`（嵐の海・濁った緑灰）＋`seaWaveStorm`（砕波 overlay）。東へ進むほど波が静まり `seaCalm` へグラデーション移行。
- 25%縮小時: 傾いた甲板（H1）とマスト（H2）の縦シルエット、嵐雲と海（H0）の暗調コントラストで「嵐の船上」を即読させる。

## 3. 高低差設計（H0 / H1 / H2）

> 戦闘ありの連絡面だが三層を活かしてアクション立体感を出す。H1 甲板を主動線、H2 帆桁を帆柱スイングで繋ぐ戦闘用の高台、H0 は舷外の荒れる海。

| レベル | このマップでの意味 | 代表エリア |
|---|---|---|
| H0 | 荒れる海面（落下即ダメージ・歩行不可） | 甲板外周・沈没船残骸の海面 |
| H1 | 傾く甲板の主動線（東西縦断） | 中央甲板・海賊頭の縄張りエリア |
| H2 | マスト上・帆桁（帆柱スイングで到達） | 帆柱基部・帆桁上の高台エリア |

- H1 甲板と H2 帆桁の境界（昇り口）に `shipMastCleat`（帆柱基部の留具）と帆柱スイング起点（§6）を配置し段差を明示。
- H1 甲板と H0 海面の境界は全周 `shipBulwarkDamaged`＋破損手すりで縁取り、甲板側の真下行に `waDropshadow` frame3 で舷影を落とす。

## 4. 歩行範囲・導線

- 主要導線: 西端（link_tashima_koro_1 から到着・H1）→ 中央甲板を東進（H1）→ 帆柱スイング選択肢（H1↔H2）→ 海賊頭エリア（H1）→ 東端の水竜の子エリア（H1/H2）→ link_kako_ko へ。途切れず1本で通る。
- 副導線（高台攻略）: 帆柱スイングで H2 帆桁へ昇り、マスト上から海賊頭・水竜の子に対して高低差優位で攻撃する選択肢。H2 からの一方通行降下も可。
- 歩行可能＝甲板（H1）と帆桁上（H2）。歩行不可＝舷外の嵐の海（H0）。落下時はダメージ後に甲板へ戻る処理。

## 5. 敵配置（座標方針）

- **海賊頭**: 甲板中央やや西寄り（H1）。プレイヤーが東進してくる導線上に仁王立ち。周囲に手下数体を配置し包囲戦の趣。先に倒すと先進フラグ立ち。
- **水竜の子**: 東端甲板（H1/H2 境界付近）。帆桁から甲板へ飛び降りる登場演出。海賊頭討伐後にフラグが立ち出現し、ボス的な役割を担う。
- 配置座標は実装時に確定。嵐の演出として波が来るたびにNPCの足元が揺れる視覚ギミックと競合しないよう配置マージンを確保する。

## 6. ギミック

- **帆柱スイング（武器ギミック・高低差移動）**: 帆柱に張られたロープに武器（斬撃/鉤縄系 W 系統アクション）を引っ掛けてスイングし、H1 甲板から H2 帆桁へ昇降する。主動線（東西縦断）はスイング無しでも通過可能＝ギミックを進行の鍵にしない原則を厳守。高台（H2）から攻撃すると海賊頭に有利、地上（H1）に留まっても攻略可能な難度に設定。
- 嵐の波浪演出: 一定間隔で甲板に波しぶきが降り注ぐ視覚演出（プレイヤー操作・判定には影響しない）。東進するほど演出が弱まり、東端では止む。

## 8. 報酬

- **嵐水晶の欠片**: 水竜の子を倒した後、東端甲板（H1）に出現するドロップ報酬。ITEM_LIST 参照。固有コードは実装時に確定。
- 手控え帳ページ: H2 帆桁上の眺望ポイントで「嵐の海域を行く船」の風景スケッチ1ページを記録できる（収集要素・進行非依存）。

## 9. イベント

- ev: 西端入場時イベント（嵐の激化と海賊頭の出現予告。船員NPCが「あいつらが待ち構えている」と短く警告）。
- ev: 海賊頭撃破後イベント（水竜の子が帆桁から飛び降りる登場カット。EVENT_LIST 参照）。

### 10.3 DoD 5本柱への対応メモ

[MAP_DEFINITION_OF_DONE](../../../../quality/MAP_DEFINITION_OF_DONE.md) の各柱を、このマップでどう満たすか：

- **構造**: H0（舷外の嵐の海）・H1（傾く甲板）・H2（マスト上/帆桁）が全て存在。H1↔H2 境界に帆柱スイング起点＋`shipMastCleat`、H1↔H0 境界に全周 `shipBulwarkDamaged`＋舷影（`waDropshadow`）で段差を明示。
- **視認性**: 25%縮小でも暗雲・嵐色の H0 海面と木目 H1 甲板のコントラスト、縦に突き出た H2 マストシルエットで「嵐の船上戦場」を即認。東端の海色が静まる変化も縮小コントラストで読める。
- **歩行範囲**: 西端→中央甲板→東端の主動線（H1）が途切れず縦断。H2 帆桁への副導線も帆柱スイングで連結し孤立エリアなし。舷外（H0）は破損手すりで歩行不可を明示し落下ダメージ処理。
- **主役と背景**: 主役＝傾く甲板（H1）とマスト帆桁（H2）の立体戦場 ／ 背景＝嵐の海（H0）・暗雲・沈没船残骸。甲板を濡れた木目の前景、嵐の海を濁緑灰の背景に統一。
- **タイル境界**: `shipBulwarkDamaged` で甲板/海を一線処理。波しぶき overlay は甲板タイルの輪郭を侵食しない透過度に設定しノイズを抑える。東端の海色グラデ（嵐→静穏）は overlay レイヤで甲板タイルと干渉させない。
