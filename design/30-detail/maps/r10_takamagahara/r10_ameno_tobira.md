## 1. 地形レイアウト方針

天の扉前は、高天原の玄関口として天之磐座(あまのいわくら)への封印扉が聳える野外広場である。マップの感情的核心は「白金×雲海の白に染まる薄明の空の聖域に、巨大な鳥居と封印された扉がただ一点の緊張を作り出す」対比にある。南から北へ向かうに連れて地盤が段階的に高まり、H0 の雲板基底面→H1 の参道広場→H2 の扉台座という三段構成を取る。H2 の中央に巨大な封印扉（天之磐座の入口）が配置され、鳥居がその手前に立つ。封印扉は W8 錫杖を用いるまで通行不能な進行ゲートであり、W8 未入手のプレイヤーはここで強制的に進行が止まる設計。高天ノ京の大門(r10_takama_kyo_mon)への経路はこの扉の解除後に北端から開通する。南側の r10_kumogaki 方向は常時通行可能で、レベルや準備が足りない段階では引き返せる。天光の金(#E8C830)の朝の光が白金砂と鳥居を縁取り、封印扉の巨大さと重さを強調する。

## 2. タイル方針

- **雲板基底床(H0)**: kaRock（明色・白金調）で雲の上の固い大地を表現。白金砂(kaSand 明色)を散点配置してH0外縁に砂浜状の余白を出す
- **参道広場床(H1)**: kaStone（明色・白金方石）で整地された参道面を形成。中央軸に kaDecor（天光の金 #E8C830）の石畳ラインを敷いて進行方向を示す
- **扉台座(H2)**: kaShrineWall（明色・社殿調）で封印扉の台座・基壇を表現。kaDecor（鳥居支柱・金属装飾）を H2 上に配置
- **巨大鳥居**: kaDecor（鳥居・朱 #C84820）を H1 北端〜H2 境界に対称配置（左右各2列）
- **封印扉**: kaShrineWall（最明・封印輝き）を H2 中央に集中配置。ギミック未解除時は kaDecor（封印紋）を重ねる
- **白金砂**: kaSand（明色）を H0 外縁と H1 際に薄く敷いて「空の砂原」感を出す
- **段差境界**: waCliff / waStairs で H0→H1→H2 を処理
- **真下影**: waDropshadow frame3 を擁壁前面行の真下セルに敷く
- **南端入口**: H0 の雲板面から H1 参道への上がり口。waStairs + waDropshadow frame3 で処理。r10_kumogaki からの登り口として視認性重視

## 4. 歩行範囲・導線

**ゲート解除前(W8錫杖未入手時):**

1. **南端(r10_kumogaki 出口)→H1 参道広場**: 南から入ると H1 に上がる。H1 全域を自由に歩行可能。
2. **H1 内の敵との交戦**: enm_072 天狗・enm_058 天の岩戸番が H1 北寄りに配置。倒せるがゲートは開かない。
3. **H2 封印扉に接触**: 封印扉に触れると「W8 錫杖がなければ道は開かぬ」テキストが出て北端への進行がブロックされる。H2 には上がれるが扉は通過不可。
4. **引き返し**: r10_kumogaki へ戻ることで準備を整えて再挑戦できる。

**ゲート解除後(W8錫杖入手・結界解除済み):**

1. **H2 封印扉→北端大門**: 扉が開き、r10_takama_kyo_mon への接続が有効になる。北端出口から大門マップへ進める。
2. **以降の往来**: FLAG_TOBIRA_KEKKAI=true が立った後は北端出口が常時通行可能になる。

## 6. ギミック

### 6-1 ★W8錫杖による結界解除進行ゲート（必須・進行制御）

- **概要**: 北端の封印扉(H2中央)はW8錫杖を所持していなければ解除できない。触れると専用テキスト「天之磐座の封印が押し返す。錫杖の力が必要だ」を表示し通行をブロック。
- **解除手順**: プレイヤーがW8錫杖所持状態で封印扉に接触→「錫杖を掲げる」選択肢→演出(扉が白光を放って開く)→FLAG_TOBIRA_KEKKAI=true セット。
- **未解除時の挙動**: H2 台座には上がれるが扉オブジェクトは通過不可。北端出口(r10_takama_kyo_mon)へのワープが無効化されている。
- **ゲームデザイン上の意味**: W8 錫杖は前工程で入手必須の鍵武器。この進行ゲートにより「錫杖を入手せずに高天ノ京へ進む」バグ的抜け道を封じる。W8 未入手プレイヤーはここで自然に引き返しを促される。

### 6-2 enm_058 天の岩戸番の縛り攻撃とW8連携

- 天の岩戸番が使う「縛り」は、プレイヤーを行動不能にする拘束攻撃。W8錫杖の「解縛の一撃」で即解除できる。
- W8所持プレイヤーへの「錫杖が実戦でも使えること」の気づき体験として機能する。ギミック解除前のチュートリアル的役割。

### 6-3 enm_072 天狗の突風・鈍足と射撃対処

- 天狗の突風攻撃はプレイヤーを後退させ鈍足状態にする。H2 台座の段差際で食らうと H1 に落とされる危険。
- W5和弓での遠距離先制が有効。W7舞扇嵐払いで突風そのものを打ち消せる。「地形の段差を活かした距離管理」が本エリアの小戦術。

### 6-4 オートセーブ（封印扉解除後）

- FLAG_TOBIRA_KEKKAI=true セット直後にオートセーブ発動。高天ノ京に進む前の確実なセーブポイントとして機能。
