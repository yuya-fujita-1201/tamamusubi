## 1. 地形レイアウト方針

金的の峯外は高天ノ京の外壁(kaShrineWall白)沿いを走る崖道マップである。北側は白壁が続き、南側は断崖の向こうに雲海(H0落下ゾーン)が広がる。崖道(H1)は西端・天ノ船着き場(r10_ameno_fune_ko)から東端・高天ノ京 大門(r10_takama_kyo_mon)まで一本道で結ばれており、大門へ向かう「寄り道ルート」として機能する。道中には外壁突出部を利用した見張り台(H2)が2か所あり、天の岩戸番が陣取っている。天狗は崖道上を飛び回り、突風でプレイヤーを崖外(H0)へ吹き飛ばす固有ギミックを持つ。薄明〜朝の光が外壁の白と雲海の白を輝かせ、天光の金が段差縁に反射する配色が基調。全体として「天上の辺境道、踏み外せば雲海の底」という緊張感と開放感の両立を狙う。

---

## 2. タイル方針

- **崖道床(H1)**: kaRock明色（白金がかった岩肌）で崖道の歩行面を形成。白金砂(itm_114)が散らばるさまを kaSand明色プロップで表現
- **見張り台(H2)**: kaRock明色やや濃いめ＋外壁接続面に kaShrineWall(白)を並べ、台上にいる天の岩戸番の存在感を高める
- **外壁(北側壁面)**: kaShrineWall(白)を H1 道面より1段高い壁として北縁全体に配置。高天ノ京の城壁感を演出
- **雲海(H0南縁)**: kaRock最暗（崖底）または空色の最淡色で雲海落下ゾーンを表現。視覚的に「奈落」とわかるよう H1 との段差を明確化
- **段差境界**: waCliff / waStairs で H0→H1・H1→H2 の全境界を処理
- **真下影**: waDropshadow frame3 を擁壁前面行（H1南縁崖・H2外縁）の真下セルに敷く
- **白金砂プロップ**: kaSand明色の散在配置を崖道各所（H1）に。itm_114 白金砂の採取ポイント演出
- **天光差し込み**: 薄明〜朝固定のため、外壁の継ぎ目や見張り台縁部に天光の金(エフェクト層)を重ねる（実装時ライティング）

## 3. 高低差設計

| 高さ | 意味 | 代表エリア |
|------|------|-----------|
| H0 | 雲海・落下ゾーン | 崖道南側の断崖下。落下ダメージ（即死ではなく強ダメージ＋崖道端に戻される） |
| H1 | 崖道・主歩行路 | 西端から東端まで続く一本道。崖道の大部分 |
| H2 | 見張り台・外壁突出部 | 外壁の突出箇所2か所（x=10〜14 y=3〜7 / x=27〜31 y=3〜7）。天の岩戸番の陣地 |

H0→H1の境界: 崖道南縁の断崖。waCliff + waDropshadow frame3。落下はダメージ＋崖道縁へ引き上げ。天狗の突風で H0 方向へ吹き飛ばされるギミックの主舞台。
H1→H2の境界: 見張り台への段差。waStairs + waDropshadow frame3。台上の天の岩戸番に接近するには段差を登る必要がある。
全体の北縁は kaShrineWall で塞がれており歩行不可（外壁）。東端・西端は各ワープ口。

## 4. 歩行範囲・導線

**主導線（西→東・大門への迂回路）:**

1. **西端入口(r10_ameno_fune_ko 方向)**：x=0〜2 の H1 崖道からスタート。道幅は約4〜6タイル（南が雲海H0、北が外壁）
2. **見張り台A(x=10〜14, y=3〜7)**：H2 見張り台。waStairs で登れる。天の岩戸番×1が守る。素通りも可能だが天の岩戸番の妨害攻撃(縛り)を受ける
3. **中央崖道(x=15〜26)**：最も道幅が狭い区間（H1幅=約3〜4タイル）。天狗が崖沿いを飛び回り、突風ギミックが最も危険な地帯
4. **見張り台B(x=27〜31, y=3〜7)**：H2 見張り台。天の岩戸番×1。東端大門が眼前に見える位置
5. **東端出口(r10_takama_kyo_mon 方向)**：x=39〜41 の H1 崖道から大門へ。大門への迂回路として完結する

**迂回路としての意義:** 高天ノ京への正規ルートとは別に、崖道を踏破することで大門に抜けられる。先を急ぐ場合は天の岩戸番を回避しながら崖道を走り抜け、天狗の突風だけをかわす判断が求められる。

## 6. ギミック

### 6-1 天狗の突風：崖外吹き飛ばし
- enm_072 が W7舞扇嵐払いを使用すると、前方扇形3タイルのプレイヤーを H1→H0 方向（南）へ2タイル強制移動させる。
- 崖道南縁(H1南端1〜2タイル以内)で突風を受けると H0 雲海へ落下し、落下ダメージ＋崖道端(H1南縁)に引き戻し。
- **対策**: 崖道の北寄り（外壁際）を歩く・突風を読んで縦回避・W5和弓で天狗に先手を打ち距離を詰めさせない。
- **中央崖道(x=15〜26)** は道幅が最も狭く(H1幅=3〜4タイル)、突風吹き飛ばしが最も危険。見張り台Aと見張り台Bの間が山場。

### 6-2 天の岩戸番の縛り×突風の複合
- H2見張り台上の enm_058 が W2木槌で縛り状態を付与した直後、上空の enm_072 が W7舞扇嵐払いを使うと、縛られたまま崖外へ吹き飛ばされる最悪の複合が発生する。
- 縛り状態中は回避行動ができないため、見張り台の天の岩戸番を先に倒す・または見張り台を大きく迂回して縛りを回避してから天狗に挑む判断が問われる。

### 6-3 白金砂の採取
- 崖道各所の kaSand明色プロップ付近で調べると itm_114 白金砂を採取可能（1〜3回。復活なし）。
- 報酬目的のプレイヤーは崖道を丁寧に踏破する動機が生まれる。急ぎたいが砂も欲しい、という選択をさせる設計。

### 6-4 見張り台の制高点
- H2 見張り台に登ると東西に広がる雲海と高天ノ京の外壁が一望できる（景観演出。§9 ローカル演出で記録）。
- 台上では enm_058 が縛り木槌を使うが、倒した後は安全地帯として機能。天狗の射程外になるタイミングもある。
