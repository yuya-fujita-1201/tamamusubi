## 1. 地形レイアウト方針

主役は「高い石壁に囲まれた番鬼専用の大広間と、天井から吊るされた鎖で区画された闘技場」。砦最奥の番鬼の間は、地下牢の廊下を抜けた先に開ける広大な石造りの大広間として設計する。マップ中央が boss_05 の初期配置となる「戦壇（H2）」で、その外周を「石床広間（H1）」が囲み、さらに外縁には「石溝（H0）」が走る。天井から吊るされた複数の鎖が足場を支えており、W5 和弓でその鎖を射抜くと足場が崩落し boss_05 が H1 に落下して第2相移行となる。背景は石造り高壁・武具の飾り立て・大型松明の赤橙光。

## 2. タイル方針

- 地面: `kaStoneFloor`（厚い石床）H1 主広間。戦壇（H2）は `kaShrineWall` 段上がりの石段＋平面に `kaStoneFloor` を敷く。
- 石溝（H0）: `kaGroove`（石溝タイル）または `kaWater_dark`（暗溝）で外縁の浅い溝を表現。踏むと移動速度低下（縛り判定はなし）。
- 壁: `caveWall`（岩壁）＋ 北壁に `kaWeaponRack`（武具掛け壁面プロップ）を配置。大型の砦内装。
- 鎖足場: `kaProp_kusari`（鎖プロップ）を天井から戦壇 H2 の四隅に垂らす。H2 戦壇はこの鎖に支えられている演出（破壊前は安定している）。
- スイッチ: `kaProp_switch`（遠隔スイッチプロップ）を戦壇 H2 の奥壁（北端高所）に設置。W5 和弓で射抜くとスイッチ ON → 鎖が一本外れ → 足場が傾き → boss_05 が H1 に落下する演出。
- 松明: `torchProp_large`（大型松明）を四隅に配置。番鬼の間は他の地下区画より明るく威圧感を強調。
- 式神陣: `kaProp_shikigami_circle`（床の術陣プロップ）を広間北側に配置。enm_021 の出現演出と連動。
- 25%縮小時: 外縁の石溝ライン・中央戦壇の段差シルエット・大型松明の橙点が読め、「ボス大広間」と即認できる構造。

## 3. 高低差設計（H0 / H1 / H2）

> 鎖スイッチギミックが高さ構造に直結。H2 戦壇の崩落が第2相移行の核心。

| レベル | このマップでの意味 | 代表エリア |
|---|---|---|
| H0 | 外縁の石溝・低下区画 | 大広間外周の石溝帯・boss_05 第2相の突進後スタン区（転落後） |
| H1 | 主広間・主戦闘床 | 入口廊下からの主広間全域・boss_05 第2相の主戦闘場 |
| H2 | 中央戦壇（鎖支持） | boss_05 の初期配置・スイッチ射抜き前の「高台」。第1相の戦闘上段 |

- H1→H0 は外縁石溝への踏み外し（移動速度低下のみ。縛りなし）。boss_05 の突進攻撃で弾き飛ばされると石溝に落ちやすい設計。
- H1→H2 は戦壇前の石段（正面1か所・側面2か所）。boss_05 は H2 から H1 への打ち下ろし攻撃を持つ。
- H2→H1 の崩落: W5 和弓でスイッチを射抜くと演出 → 鎖が切れ → H2 戦壇が傾き → boss_05 が H1 に落下（第2相移行）。プレイヤーは先に H1 に退いてから射抜く設計（自分も一緒に落ちないよう）。
- 全境界に `kaShrineWall` または `kaDote` を配置（HEIGHT チェック対応）。

## 4. 歩行範囲・導線

- 入口: 西端（H1）廊下接続ワープ → 大広間（H1）に出る。
- 広間 H1 から中央戦壇 H2 へは正面石段（南側1か所）・側面石段（東・西各1か所）の3ルート。
- 戦壇 H2 奥の北端高所にスイッチがある。H2 に上がり奥に進んで「スイッチを発見」した後、一度 H1 に退いてから W5 和弓で射抜く流れが自然な動線。
- 石溝 H0 は外周を一周できるが、戦闘中に踏み外すと移動速度低下が不利に働く危険地帯。
- 出口はなし（boss_05 撃破後イベントで r3_chika_sekisho に戻るワープ発生）。

## 6. ギミック（ボス2相構成詳述）

### boss_05 関守の番鬼 — 第1相

- 初期配置: 中央戦壇 H2 に仁王立ち。大太刀を持ち、正面方向へのガード（正面攻撃無効）が特徴。
- 攻撃パターン:
  - 「大太刀薙ぎ」: H2 壇上で横なぎ。H1 石段を上がってきたプレイヤーを巻き込む広範囲。
  - 「地割れ突進」: H2 から H1 へ飛び降り突進。着地点を中心に H1 が揺れ、周囲 3 マスにダウン効果。
  - 「鉄壁ガード」: 正面からの攻撃を完全反射。背後・側面を狙う必要がある。
- **第2相移行条件**: 戦壇 H2 奥のスイッチを W5 和弓で射抜く（HP 残量に関係なく任意タイミング可）。 → 「鎖が切れる SE」→ H2 戦壇が傾く → boss_05 が H1 へ落下。

### boss_05 関守の番鬼 — 第2相

- H1 主広間に落下した boss_05 は激怒モードで行動速度と攻撃範囲が増大する。
- **鉄壁ガードが一時的に解除**（落下の衝撃でバランスを崩したため）→ 正面からの攻撃も有効な短時間ウィンドウが発生（炒り豆 itm_014 を撒くとさらに怯みが延長）。
- 追加攻撃:
  - 「連続溜め突進」: H1 広間を左右に突進。石溝 H0 の壁に激突してスタン（プレイヤーが H0 端に誘導して時間を稼ぐ戦術）。
  - 「激怒震脚」: 足踏みで H1 全体に低衝撃波。縛り効果はないが移動方向が乱れる。
- **勝利条件**: boss_05 本体の HP を0にする（正面無効は第2相では一時解除中）。W2 木槌の鈍器が弱点で大きなダメージ倍率。
- W5 和弓の使い方: スイッチ射抜きという「弓の練習」として機能。遠方の目標に命中させる感触をここで初体験させ、後続の r3_chika_kura での遠隔鍵機構解除へ繋げる。

### その他ギミック

- **武具掛け調べ**: 北壁の `kaWeaponRack` を調べると「関所警備の規律と武具の種類」フレーバーテキスト。boss_05 の背景補足。
- **スイッチ到達ヒント**: 初入場時、H2 奥のスイッチに近づくと「何かを射抜けば作動しそうなスイッチがある」テキスト。W5 和弓を持っていれば「和弓で狙えるかもしれない」に変化。

### 10.3 DoD 5本柱への対応メモ

[MAP_DEFINITION_OF_DONE](../../../../quality/MAP_DEFINITION_OF_DONE.md) の各柱を、このマップでどう満たすか1行ずつ：

- **構造**: H0（石溝外縁）・H1（主広間）・H2（中央戦壇）の三段が全て存在し、全境界に `kaShrineWall`/`kaDote` を配置して HEIGHT チェックに対応する。
- **視認性**: 25%縮小で外縁石溝の暗帯・中央戦壇の段差・大型松明の橙点が読め、ボス大広間と即認できる。
- **歩行範囲**: 入口廊下→主広間→石段3ルートで戦壇→北端スイッチまで途切れず通じ、W5 和弓ギミックに必要な「射線が通る距離」が確保されている。
- **主役と背景**: 主役＝中央戦壇・鎖・スイッチの構造物と boss_05 ／ 背景＝石造り高壁・武具掛け・式神陣の優先順位。
- **タイル境界**: `kaStoneFloor`（H1/H2）/ `kaGroove`（H0）/ `kaShrineWall`（段差）を明確に分割し、ボス戦中に H0 危険帯・H1 主戦場・H2 壇を瞬時に把握できる視認性を確保する。
