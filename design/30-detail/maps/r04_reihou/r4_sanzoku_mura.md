## 1. 地形・町割り

H1 の岩棚（中腹）に張り付くように形成された小集落。崖側（北）を背にして南向きに家屋が連なり、山師・行者・稜線往来の旅人が立ち寄る唯一の宿場町として機能する。R4 全域において W 系統の上位強化が最初に解禁される重要拠点——R5 装備底上げの起点。

町割りは「岩棚の一本街道」型。西口（r4_takazan_ichi 側）から東口（r4_kanko_michi 側）へ抜ける一本道（H1）の両脇に、板屋根の家屋が 3〜4 軒ずつ並ぶ。道の中央付近に鍛冶場（kaKajiba）を配置し、武器強化の煙と火の光が集落の中心目印となる。北側崖下に宿の暖簾（kaYadoNoren）を吊るした休憩小屋。南側は崖の縁のため低い石垣（kaIshigaki）で転落防止。

## 2. タイル方針

- 屋根群: `kaItayane`（板屋根）を集落の家屋に使用。山岳集落の質朴な木板葺き。
- 岩棚地面: `kaIwadanaShuraku`（岩棚集落地盤）で H1 の岩棚面を表現。石畳ではなく粗い砂利＋岩盤。
- 鍛冶場: `kaKajiba`（鍛冶場）を集落中央道沿いに配置。煙突プロップ（`kaProp_entotsu`）を添える。
- 宿: `kaYadoNoren`（宿暖簾）を北側小屋入口に吊るす。行灯（`kaProp_andon`）を軒下に。
- 段差・石垣: 南縁の転落防止に `kaIshigaki`（石垣）。西口/東口の坂道段差に `kaIshiDan`（石段）。
- 影: 各家屋・石垣の前面下セルに `waDropshadow` frame3（影）を敷き、岩棚の立体感を強調。
- 色彩（R4 §2 H1 岩棚集落): 灰褐（#6A6A7A 系）の岩盤地盤＋板屋根の茶灰＋鍛冶場の橙火点光源。25%縮小でも板屋根群と鍛冶場の煙突で「山中の集落」と即認できる。

## 3. 高低差設計（H0 / H1 / H2）

| レベル | このマップでの意味 | 代表エリア |
|---|---|---|
| H0 | 西口・東口の坂道入口 | 西端 r4_takazan_ichi 接続部・東端 r4_kanko_michi 接続部 |
| H1 | 岩棚の町本体（一本街道・施設群） | 鍛冶場・宿・商店・NPC 配置エリア全体 |
| H2 | 北側の崖上縁（展望テラス） | 北崖縁の小展望台（フレーバーポイント） |

- H0→H1: 西口/東口それぞれに `kaIshiDan`（石段）2〜3 段。坂を登り切った先が町の入口。
- H1→H2: 北崖側の石段（`kaIshiDan`）で崖縁の展望テラスへ。行き止まり（歩行不可の岩壁 `kaIwaKabe` で閉じる）。
- 各段差前面行の真下セルに `waDropshadow` frame3。

## 4. 歩行範囲・導線（施設入口配置）

- 主要導線: 西口（H0→H1石段）→一本街道（H1）→鍛冶場前（H1・強化 NPC）→宿入口（H1 北側）→東口（H1→H0石段）が途切れず通じる。
- 施設入口:
  - **鍛冶場**: 街道中央、南向き入口。強化 NPC「鍛冶の清右衛門」が外に立つ。
  - **宿**: 北側崖下の小屋。入口（H1）に暖簾。女将 NPC が帳場に立つ。
  - **雑貨売り**: 鍛冶場東隣の小店。消耗品 NPC が店頭に立つ（itm_069 火打石・itm_014 炒り豆）。
  - **展望テラス**: 北崖側石段（H1→H2）を登った突端。フレーバー調査ポイント。
- 崖縁（南側石垣外）・家屋内部（壁内）・北崖岩壁は歩行不可。

## 5. 敵配置

— 町内は非戦闘エリア（安全圏）。enemySpawn なし。町境（西口 r4_takazan_ichi 側・東口 r4_kanko_michi 側）の野外マップに戻ると敵が出現する。

## 6. ギミック・施設

- **武器強化（鍛冶場）**: 鍛冶の清右衛門 NPC に話しかけると W 系統の上位強化レシピが解禁（R4 初解禁）。対象武器: W1 祓い刀 / W2 木槌 / W3 御鏡 / W5 和弓 / W6 式符 / W7 舞扇。素材として itm_070 霊砂 + itm_069 火打石が必要なレシピが中心（詳細レシピは後続設計で確定）。
- **宿・セーブ**: 女将 NPC に話しかけ「お宿に泊まりますか？」→HP/SP 全回復＋セーブ確定。
- **雑貨売り**: itm_069 火打石・itm_014 炒り豆・itm_097 霊峰の登拝札（フラグ条件付き）を販売。
- **展望テラス（H2）**: 北崖縁の調査ポイント。「霊峰の白い頂が雲の上に突き出している。ここより上は別の世界だ。」フレーバーテキスト。
- **清右衛門の情報 NPC**: W 上位強化解禁後に「霊砂を多く使うが、R5の鬼には霊砂を混ぜた刀でないと傷一つ付かぬ」と情報提供（R5 への伏線）。

### 10.3 DoD 5本柱への対応メモ

[MAP_DEFINITION_OF_DONE](../../../../quality/MAP_DEFINITION_OF_DONE.md) の各柱を、このマップでどう満たすか1行ずつ：

- **構造**: H0（坂道入口）/ H1（岩棚の町本体）/ H2（北崖縁展望テラス）の三段が全て存在し、各境界に kaIshiDan と waDropshadow を配置する。
- **視認性**: 25%縮小でも板屋根群（kaItayane）の連なりと鍛冶場の煙突（kaProp_entotsu）が「山中の集落」として読める。
- **歩行範囲**: 西口から東口まで H1 の一本街道が途切れず通じ、鍛冶場・宿・雑貨売りへの入口が街道から自然にアクセスできる。
- **主役と背景**: 主役＝鍛冶場の橙火と清右衛門 NPC（W 上位強化 UI 起点）／背景＝板屋根群・岩棚地盤・北崖の展望テラスの優先順位。
- **タイル境界**: 岩棚地盤（kaIwadanaShuraku）・板屋根（kaItayane）・石段（kaIshiDan）・石垣（kaIshigaki）を明確分割し、H0/H1/H2 の高度差を直感的に読ませる。
