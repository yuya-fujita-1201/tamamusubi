## 1. 地形レイアウト方針

主役は「干した漁網が島全体を覆う、生活感あふれる小島」。島中央の高台（H2）に干し場が広がり、縦横に張られた漁網がプロップとして並ぶ。北岸と南岸に舟着き場（H1 砂浜）があり、島周辺は浅い瑠璃の内海（H0）に囲まれる。中央の干し場下に隠し通路が存在し、干した漁網（`fishingNet` プロップ）を動かすことで入口が露出する。背景は穏やかな内海と遠景の小島シルエット。主役は網と高台干し場（H2）、背景は砂浜（H1）と海（H0）。

## 2. タイル方針

- 海面（H0）: `calmSea`（穏やかな瑠璃色）。島周囲の浅瀬は `shallowSea`（薄い水色）で H0/H1 の遷移を表現。
- 砂浜（H1）: `sandShore`（白砂）。舟着き場には `woodenPier`（木桟橋）を追加。
- 高台干し場（H2）: `driedEarth`（土間）+ `fishingNetProp`（干した漁網・展開プロップ）。網の列をH2全体に格子状に配置。
- 隠し通路壁: `stoneWall`（石組み）。漁網プロップが前面を覆い通常時は識別不能にする。
- 隠し祠接続: `hiddenPath`（苔生した石段）で H2 から r2_amijima_hokora への内部連絡。
- 25%縮小時: 島シルエットと網の格子パターンで「漁師の小島」を即認させる。干し場の H2 高台が明確に浮かぶこと。

## 3. 高低差設計（H0 / H1 / H2）

> 三層構成。H0 = 海面、H1 = 砂浜・舟着き場、H2 = 島中央高台（干し場）。

| レベル | このマップでの意味 | 代表エリア |
|---|---|---|
| H0 | 内海海面（瑠璃・穏やか） | 島周囲の海全域 |
| H1 | 砂浜・浅瀬・舟着き場 | 北岸・南岸の砂浜、舟着き場（桟橋） |
| H2 | 島中央高台（干し場・漁具置き場） | 島中央の土間エリア、隠し通路入口付近 |

- H0/H1 境界: `tidalEdge`（砂浜縁）で明示。波打ち際表現（HEIGHT チェック対応）。
- H1/H2 境界: `sandStep`（砂混じり石段）で全境界に段差を配置。
- 擁壁前面行の真下セルに `waDropshadow` frame3 を敷く。

## 4. 歩行範囲・導線

- 主要導線: 北岸舟着き場（H1）→ 石段で H2 高台へ → 南岸舟着き場（H1）まで通り抜けられる。または東岸の r2_minato ワープ口（H1）へ直接。
- H2 高台は全域歩行可。漁網プロップは当初「行く手を一部塞ぐ壁状配置」でギミックを構成する。
- 隠し通路: H2 中央南側の網群を動かすと石段が出現し、r2_amijima_hokora（祠マップ）への内部ワープ口に繋がる。
- H0 海面は不歩行（舟移動演出）。H1 砂浜は歩行可。

## 6. ギミック

- **漁網移動ギミック（メイン）**: H2 中央南側の `fishingNetProp`（干した網・移動可能オブジェクト）3 組を、各組をアクションキーで押し/引きして所定の位置に移動させると、背後の `stoneWall` が実はダミーであることが判明し `hiddenPath` が出現。アイテム不要・武器不要（誰でも操作可）。
- **W2 木槌連携**: enm_011 大蟹の甲羅を砕くと甲羅が床に残り、漁網ギミックの「最後の台座」として機能する（任意強化ルート）。殻なしでも手順を変えてクリア可能。
- **itm_066 潮塩・itm_065 真珠の産地**: H1 砂浜の特定セルに固定配置（各2個）。探索動機として機能。
- 鍵・照明を進行の鍵にしない原則を守る。

### 10.3 DoD 5本柱への対応メモ

[MAP_DEFINITION_OF_DONE](../../../../quality/MAP_DEFINITION_OF_DONE.md) の各柱を、このマップでどう満たすか：

- **構造**: H0（内海）/ H1（砂浜）/ H2（高台干し場）の三層が全て存在し、全境界に `tidalEdge`（H0/H1）・`sandStep`（H1/H2）を配置。
- **視認性**: 25%縮小でも島シルエット・格子状漁網パターン・H2 高台が「漁師の小島」として識別可能。
- **歩行範囲**: 北岸 H1 → H2 高台 → 東岸 H1 → 南岸 H1 が途切れず通る。隠し通路は漁網ギミック後のみ追加。
- **主役と背景**: 主役＝H2 高台干し場＋漁網プロップ ／ 背景＝H0 内海・H1 砂浜。高台をパレット（乾いた土色）と高さで前景に優先。
- **タイル境界**: `tidalEdge` で H0/H1 を、`sandStep` で H1/H2 を一線処理。漁網 overlay は H2 地面タイルを完全に隠さない不透明度に設定。
