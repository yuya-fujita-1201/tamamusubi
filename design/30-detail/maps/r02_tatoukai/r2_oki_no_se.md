## 1. 地形レイアウト方針

主役は「嵐の予兆と荒波が支配する外縁の岩礁群」。内海の穏やかな青に対して、沖の瀬は白波が絶えない荒れた水面（濃紺・灰白）を敷き、遠景に黒雲と稲光の背景エフェクトを重ねる。岩礁が散点的に H1/H2 として突き出し、航路は細く曲がりくねる。マップ東縁は外洋の開けた広がりで、「もうすぐ大海に出る」緊張感を演出する最前線のエリア。背景は荒れた外洋（H0）と遠景黒雲で、点在する岩礁と波頭が主役として視覚を引く。

## 2. タイル方針

- 海面（H0）: `roughSea`（荒波・濃紺）系。波アニメーションを `seaWave` より速いフレームレートで設定。白波 `whitecap` をランダムに散在。
- 浅瀬岩礁（H1）: `roughRock`（荒岩）系。白波が岩に砕ける縁に `waveBreak` スプライトを配置。
- 高岩（H2）: `seaStack`（海食柱）系。数本の細い柱状岩で H2 を表現し、海鳥の巣・海賊崩れの見張り台として機能。
- 嵐演出: `stormCloudOverlay`（嵐雲エフェクト）を最上層にオーバーレイ。ev_118「嵐の前触れ」フラグで 3 段階変化（晴れ→曇り→嵐）。
- 25%縮小時: 荒岩群のシルエットと濃紺の波が「危険な外縁海域」を即認させる。

## 3. 高低差設計（H0 / H1 / H2）

> 海面基準三層。H0 = 荒れた外洋水面、H1 = 浅瀬・岩礁・砂礫帯、H2 = 高岩（海食柱・見張り台）。

| レベル | このマップでの意味 | 代表エリア |
|---|---|---|
| H0 | 荒れた外洋水面（白波散在） | マップ全域の水面・航路 |
| H1 | 浅瀬岩礁・砂礫帯（波が砕ける縁） | 中央岩礁群、北岸浅瀬帯、南端砂礫列 |
| H2 | 高岩・海食柱（見張り台・巣） | 東寄りの突出岩3本、北東の海賊崩れ拠点岩 |

- H0/H1 境界: `roughRockEdge` + `waveBreak` で波が砕ける表現を配置（HEIGHT チェック対応）。
- H1/H2 境界: `seaStackBase`（海食柱根本）で明確な段差を表現。
- 擁壁前面行の真下セルに `waDropshadow` frame3 を敷く。

## 4. 歩行範囲・導線

- 主要導線: 西端ワープ口（H1 岩礁縁）→ 岩礁伝いに東進 → 南端 r2_nango_se ワープ口（H1）まで一本道（曲折しながらも途切れない）。
- H2 高岩への上昇: 北東の海賊崩れ拠点岩へは H1 から石段（1 列分）で登れる。enm_015 海賊崩れの根城で手控帳ページを入手できる。
- 嵐フラグ「嵐の最高潮」時: H1 砂礫帯の一部が波に洗われ不歩行化（ev_118 の環境変化）。迂回路は常に確保。
- H0 水面は不歩行。岩礁（H1/H2）のみを伝い歩きする島渡り感を演出。

## 6. ギミック

- **嵐 3 段階演出（ev_118）**: ev_118「嵐の前触れ」環境イベントによる段階変化。晴れ→曇り（`stormCloudOverlay` 半透明）→嵐（`stormCloudOverlay` 全面＋波アニメ加速＋一部 H1 不歩行化）。進行不能にはならない設計。
- **海賊崩れの拠点**: H2 高岩上に木箱・錨等のプロップ。enm_015 全滅後に手控帳ページ「海賊崩れの顛末」が開錠。アイテムを進行の鍵にしない原則を守る（任意探索）。
- **W4 薙刀「潮薙ぎ」**: r2_toiya で入手する W4 薙刀がある場合、H0 縁の enm_012 近海鮫を遠距離で払える。薙刀なしでも H1 から迂回可能なルートを必ず用意する。

### 10.3 DoD 5本柱への対応メモ

[MAP_DEFINITION_OF_DONE](../../../../quality/MAP_DEFINITION_OF_DONE.md) の各柱を、このマップでどう満たすか：

- **構造**: H0（外洋）/ H1（岩礁）/ H2（高岩）の三層が全て存在し、全境界に `roughRockEdge` または `seaStackBase` を配置。
- **視認性**: 25%縮小でも荒岩群のシルエットと H2 海食柱が外縁危険地帯として識別可能。嵐 overlay は H1 岩礁を隠さない透過度に制限。
- **歩行範囲**: 西端 H1 → 中央岩礁伝い → 南端 H1 が途切れず通る。嵐最高潮時も迂回路で到達可能。
- **主役と背景**: 主役＝荒岩群（H1/H2）と白波 ／ 背景＝濃紺外洋（H0）と遠景黒雲。岩礁を高さとパレットで必ず前景に出す。
- **タイル境界**: `roughRockEdge` + `waveBreak` で H0/H1 境界を波砕け表現で統一。H2 柱根本は `seaStackBase` で一線処理。ノイズ（無意味な 1 セル突起）は排除。
