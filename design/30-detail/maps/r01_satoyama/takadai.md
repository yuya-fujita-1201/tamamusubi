## 1. 地形レイアウト方針

石垣と杉林に囲まれた急斜面の高低差地形が主役。実装は「不透明3D石垣（waCliff3）で高低差を表現」「前面=高い玉石壁3段 / 中央=石段 / 下段=暗い草地」（第2版実装コメント）。下段草地（H0）→石段（H1遷移）→上段高台草地（H2）という急勾配が視覚の核となる。見晴らし効果は H2 到達後の開けた空間で演出する。

## 2. タイル方針

実装定数 `waCliff3`（不透明3D石垣21フレーム）・`waStairs3`（不透明石段）・`waDropshadow`（擁壁影）・`grassUpper`（上段草）・`grassLower`（下段暗草）・`cliff`（鎮守の森）を全面活用。前面壁は cap→壁3段→基部の連続立体陰影グラデ、N/W/E 縁は笠石縁＋外角、角は立体的に巻く（お手本016/p01準拠）。25%縮小では石垣の縦ストライプと石段のジグザグが主役として機能する。

## 3. 高低差設計（H0 / H1 / H2）

> 実装コメント「高台(40×32)・前面壁3段擁壁」から直接読み取れる3段構成。高低差がこのマップの存在意義。

| レベル | このマップでの意味 | 代表エリア |
|---|---|---|
| H0 | 低地: 下段暗い草地・南入口 | 下段草地（y≈15〜31、石垣前面より南全域） |
| H1 | 中段: 石段・石垣前縁の遷移帯 | 石段・石垣前縁行（y≈10〜14） |
| H2 | 高所: 上段台地・杉林平場・見晴らし空間 | 上段草地（y≈0〜9、x:6〜33） |

- 前面壁（waCliff3）の基部行（BASEY=y14）の真下 `waDropshadow` frame3 を必ず敷く（DoD 高さ制約）。
- H2 上段の北縁・東縁にも笠石縁（RIM_N/RIM_E）を配置し上段の輪郭を明確化。

## 4. 歩行範囲・導線

- 主要導線: 南端入口（H0, satoyama 方面）→下段草地→石段（中央 x≈18〜22, CAPY〜BASEY 付近）→H2 上段台地→東端（tanada へのワープ）。
- 石段は中央1本のみ（急斜面感の強調）。迂回路なし。押し返す敵との戦闘で石段を守る構造になる。
- H2 上段では全方向に広い視界が生まれ「見晴らし」感を演出。杉林（cliff タイル）が東西の外縁を構成。
- H0 下段の東西縁は歩行不可（cliff 鎮守の森で挟む）。南入口からの進行方向が正面に限定される。

## 6. ギミック

**W2木槌＝もろい岩砕き**: H2 上段の北東角に「もろい岩」タイルを配置。W2 木槌（takadai 付近の里人台詞で入手誘導）で砕くと tanada への近道ショートカットが開く。W2 入手前は回り道（H0→satoyama 経由）が必要。里人台詞「その丸太槌で岩を砕いてみろ」が誘導。

## 10. map-quality パイプライン接続（必須）

### 10.1 map_goal 入力（レビュー用 目的文）

```
急斜面の高低差地形（40×32）。下段草地(H0)から中央石段(H1)を登り上段台地(H2)へ至る一本の導線が主役。主役は石垣の前面壁3段と石段、背景は下段の暗い草地と杉林。プレイヤーがW2木槌の「もろい岩砕き」ギミックを初めて体験し、tanada への道が開く。高台からの見晴らしで八洲の里山世界の奥行きを体感させる。
```

### 10.2 heightmap 入力（`checker/heightmaps/takadai.json` の設計）

> 粗い代表グリッド（5列×4行）でH0/H1/H2の分布意図を示す。実装時に正寸(40×32)へ展開。

```jsonc
{
  "map": "takadai",
  "w": 40, "h": 32,
  // levels[y][x]: 0=H0低地(下段草地/南入口) / 1=H1中段(石段遷移帯) / 2=H2高所(上段台地)
  // 代表グリッド: w=40→5分割(8/格)、h=32→4分割(8/格)
  "levels": [
    [0, 2, 2, 2, 0],  // y≈0-7  上段台地・杉林（H2）
    [0, 2, 2, 2, 0],  // y≈8-15 上段台地南部（H2）＋石段遷移（中央H1）
    [0, 0, 1, 0, 0],  // y≈16-23 石段・石垣前縁（H1）＋下段草地（H0）
    [0, 0, 0, 0, 0]   // y≈24-31 下段草地・南入口（H0）
  ]
}
```

### 10.3 DoD 5本柱への対応メモ

[MAP_DEFINITION_OF_DONE](../../../../quality/MAP_DEFINITION_OF_DONE.md) の各柱を、このマップでどう満たすか：

- **構造**: H0（下段草地）・H1（石段遷移）・H2（上段台地）の3段が全て存在し、石垣前縁行に waDropshadow frame3 を敷く（実装コメント BASEY=y14 準拠）。
- **視認性**: 25%縮小で waCliff3 の縦ストライプと石段ジグザグが構造を即読させる。上段と下段の輝度差で H 分布が明確。
- **歩行範囲**: 南端→石段→H2 上段東端（tanada）の主導線が途切れず通る。
- **主役と背景**: 主役＝石垣前面壁3段＋石段／背景＝下段暗草地・杉林。石垣の立体感が視覚の核。
- **タイル境界**: waCliff3 の21フレーム連続陰影が境界ノイズを自動処理。角は立体コーナー処理済み。
