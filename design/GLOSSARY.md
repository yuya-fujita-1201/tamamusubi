## 1. 世界・コスモロジー

| 用語 | 読み | 意味 |
|---|---|---|
| 玉結び | たまむすび | 本作タイトル。表＝勾玉を集めて結ぶ／裏＝魂結び（さまよう魂を結び留める民俗儀式）の二層設計。英語表記 TAMAMUSUBI |
| 八洲 | やしま | 世界名。記紀「大八洲」由来。地上・海の彼方・雲海の上の**三層**からなる擬似日本列島世界 |
| 三層構造 | — | 地上（列島）／海原（常世の浜）／天空（高天原）。別ワールドではなく**1つの連続世界の層**。移動手段（徒歩→船→飛空艇）で層が開く |
| 依代 | よりしろ | 各土地で神を祀る対象。光が弱まると穢れが宿る。灯し直すのが旅の主目的の一つ |
| 穢れ | けがれ | 依代の光が弱まり獣や精霊に宿る災い。敵＝穢れた魂が妖の形をとったもの（殺すのではなく**祓って還す**） |
| 八百万 | やおよろず | 八洲のコスモロジー基底。鳥居・祓い・依代を世界の魂とする（明るいトーン） |
| 産霊 | ムスヒ | 造化三神（タカミムスビ・カミムスビ）に連なる「結び」の力の語源。結び師 → 縛り祓い → 玉結びへ繋がる系譜 |
| 語られない秘密 | — | なぜ依代が弱まるのか／なぜ高天原は封じられたのか。本筋では説明せず遺構・風景・断片会話で匂わせる（テラリア/Tunic式） |

## 5. 探索・設計用語

| 用語 | 意味 |
|---|---|
| 移動の連続性 | FF6/FF9方式。マップ選択ジャンプを禁止し「同じ世界を移動している」感覚を核にする |
| 地平線の引力 | 遠景 parallax に高天原の光・常世の水平線を常に見せ「あそこは何だろう」を植え付ける設計 |
| スニーク先取り | 敵視界（sightR）外を縫って強敵地帯のレア（勾玉）を先取りするFF11式の遊び。経路は必ず複数 |
| 秘密の密度 | 3画面に1個上限（多すぎると感動が薄れる）。武器ギミック記号と共有カウント |
| 手控え帳 | 収集アイテム（ページ）。世界の断片を集める |
| HD-2D parallax 遠景 | 近景0.7／遠景0.35 の2枚重ね背景。高天原の光・常世の水平線を常時提示 |
| map-quality パイプライン | `../quality/` + `../checker/`。地形構造→導線→高低差→建物→装飾の順で作り自動検査するマップ品質保証フロー |
| 着域演出 | （§2参照）層・地域の切り替えを体感させる三点同時切り替え |
