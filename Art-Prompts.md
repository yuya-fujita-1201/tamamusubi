# Art-Prompts.md — 『玉結び』キーアート＆キャラクターアート GPT Image 2.0 用プロンプト集

> 作成: 2026-06-13 ／ 既存コンセプトアート5枚（`image/`・明るい彩度の高いアニメファンタジー画風）と調和する前提で設計。
> 生成はユーザーが GPT Image 2.0（ChatGPT）で実施。プロンプトは日本語（Concept_logs.md 時代からの実績ある方式）。

## 0. 使い方（重要・キャラ一貫性の出し方）

1. **生成順序**: まず §2「ミト キャラシート」を生成 →「正解のミト」を1枚確定 → 以降のキービジュアル生成時は**その画像をChatGPTに添付**（「添付の主人公と同じキャラクターで」と指示）。これで顔・衣装のブレを最小化
2. **貼り方**: 各プロンプトは「①共通スタイル → ②ミト設定ブロック → ③シーン本文」の三段連結。キャラ無しの風景のみなら②を省略
3. **アスペクト比**: キービジュアル＝16:9（横長）／キャラシート・立ち絵＝2:3（縦長）／スクエアはアイコン用
4. **禁止チェック**: 暗い・ホラー調になったら不採用。侍鎧・忍者装束・現代物が混入したら「除外指定」（§6）を追記して再生成
5. ロゴ・文字は基本入れない（既存アートのYASHIMAロゴは仮。タイトル『玉結び』ロゴは別途デザイン）

## 1. 共通スタイルブロック（全プロンプト冒頭に貼る）

```
明るく彩度の高いアニメファンタジーのキーアート。透明感のある美しい空と光、緻密で叙情的な背景、神話の息づく和風ファンタジー世界。色彩は豊かで開放的、ホラー要素なし、グロなし。高品質なゲームのキービジュアル用イラスト。文字・ロゴ・ウォーターマークなし。
```

## 2. 主人公ミト — キャラクター設定ブロック（正典）

### 設定ブロック（②としてキャラ登場プロンプトに必ず貼る）

```
主人公ミト: 15歳くらいの中性的な若者。墨色の短めの髪をうなじで小さく結び、朱色の組紐の髪飾り。琥珀色の瞳、意志の強い穏やかな表情。服装はファンタジー和装の旅装——白い筒袖の上衣、若草色の動きやすい袴風の脚衣、藍色の短い羽織（袖と裾に組紐の結び目飾り）、朱色の組紐の帯と襷、脚絆と旅沓。腰に「祓い刀」（白い鞘に注連縄と紙垂を結んだ祭具のような刀）、首に勾玉の首飾り（3つの勾玉）、腰に小さな手控え帳。そばに白い折り紙の鳥の精霊「シロ」（淡く霊文が光る式鳥）が飛んでいる。侍や忍者の記号は無し。明るく親しみやすいデザイン。
```

### 2-A. キャラシート（最初に生成して正解を確定）

```

このキャラクターのデザインシートを描いてください。白背景に、正面全身立ち絵・横向き全身・後ろ姿の3面図を並べ、隣に顔の表情差分（無表情・微笑み・決意）と、祓い刀と勾玉の首飾りのクローズアップを配置。ゲーム開発用のキャラクター設定資料として、デザインの細部がはっきり分かるように。
```

```
ミトが祓い刀を静かに構え、刀の注連縄と紙垂が風に揺れる縦長のキーアート。背景は朝焼けの空と遠くの鳥居のシルエット、花びらと淡い光の粒が舞う。シロが肩のそばを飛ぶ。凛として、どこか優しい眼差し。
```

## 3. キービジュアル（フィールド別・16:9）

> 各プロンプト＝①共通スタイル＋②ミト設定ブロック＋以下のシーン本文。
> 構図の原則: **ミトは画面の1/3以下のサイズで世界の広がりを主役に**（探索ゲームの「あっちに行きたい」を絵で語る）。

### 3-1. 始まりの里山郷（体験版の舞台）

```
朝の光に満ちた里山の棚田を、ミトが畦道を歩いて旅立つ後ろ姿。注連縄の大鳥居と水車小屋、桜と若葉、遠くに御神木の杜。空は澄んだ青、水田に空が映る。シロが先導するように前を飛ぶ。画面の奥、水平線の彼方にうっすら光る海と、雲間に金色の建築物の影が小さく見える。冒険の始まりの高揚感。
```

### 3-2. 霧立の里（拠点の村・夕暮れ）

```
夕暮れの山里。石灯籠に灯がともり、茅葺きの家々から夕餉の煙が上がる。ミトが里の大鳥居の前で老巫女と言葉を交わしている。温かな琥珀色の光、ホタルのような淡い光の粒。懐かしくて優しい、帰る場所の絵。
```

### 3-3. 根の洞（体験版ダンジョン・光差す洞窟）

```
巨大な木の根が絡み合う神秘的な洞窟。天井の裂け目から光の柱が差し込み、青く光る苔と水晶が闇を照らす。ミトが御鏡を掲げ、鏡から放たれた光の筋が曇った鏡石に反射して奥の祠を照らしている。怖くない、神秘的で美しい地下世界。
```

### 3-4. 瑠璃の多島海（船の旅）

```
ターコイズブルーの多島海を、白い帆の小舟でミトが渡る。段々畑の島々、白い灯台、海面に反射するまぶしい陽光。シロが帆の上にとまっている。遠くの水平線が異様に明るく金色に輝いている（常世の予感）。爽快で開放的な海の旅。
```

### 3-5. 宮乃京（古都の祭り）

```
朱塗りの大橋と五重塔のある明るい古都。提灯と色とりどりの吹き流しが揺れる祭りの大通りを、ミトが人混みの中を歩く。露店、舞う花びら、楽しげな人々と小さな善き妖たち（可愛らしい精霊）が混じって祭りを楽しんでいる。雅やかで賑やかな都の絵。
```

### 3-6. 霊峰の裾野（山岳）

```
雲がたなびく霊峰の登山道。鳥居が連なる石段をミトが登り、振り返ると眼下に雲海と里山の絶景が広がる。高山の澄んだ空気、ハイマツと岩肌、遠くに雷鳥のような白い鳥。スケールの大きな縦の景観。
```

### 3-7. 北雪郷（雪国・明るい雪景色）

```
一面の雪原と雪をかぶった合掌造りの集落。夜ではなく晴れた昼、雪の反射でまぶしいほど明るい。ミトが雪装束の羽織を重ね、白い息を吐きながら歩く。氷柱が陽光にきらめき、空にはオーロラのような淡い光の帯。寒いのに温かい、静かで美しい雪国。
```

### 3-8. 黒杉の幽谷（唯一の薄暗ゾーン・それでも美しく）

```
巨大な黒杉が立ち並ぶ深い谷。霧の中、木漏れ日が幾筋も差し込み、ミトの提灯が暖かく灯る。杉の幹に結ばれた古い注連縄、苔むした石仏たち。暗いが不気味ではなく、神聖で静謐な森の深部。光と霧のコントラストが美しい。
```

### 3-9. 常世の浜（海の彼方の楽園）

```
白珊瑚の鳥居が立つ純白の砂浜と、信じられないほど透明なターコイズの海。ミトが裸足で波打ち際に立ち、振り返って微笑む。珊瑚と貝殻の祠、砂浜に続く無数の淡い足跡、空と海の境界が光で溶けている。この世のものではない美しさ、永遠の楽園。
```

### 3-10. 高天原（雲海の上の白金の都）

```
雲海の上に浮かぶ白金の空中都市群。白い石柱と水盤、金の装飾、浮島をつなぐ光の回廊。ミトが天磐船の舳先から、目の前に広がる神々の都を見上げている。荘厳だが廃都のように静か、神聖な光に満ちている。圧倒的なスケールの絵。
```

### 3-11. 天磐船・雲海飛行（メインキービジュアル候補）

```
夕焼けと青空が混じる壮大な雲海の上を、和様の飛空艇「天磐船」が帆に風を受けて飛ぶ。舳先にミトが立ち、髪と羽織が風になびく。シロが並んで飛ぶ。眼下の雲の切れ間から明るい列島が覗き、遥か前方に高天原の光が見える。「見えていた場所に、いつか本当に辿り着く」を一枚で語る本作の象徴的キービジュアル。
```

## 4. アクションキーアート（戦闘・見せ場）

### 4-1. 祓い波（必殺技）

```
ミトが祓い刀を大きく振り抜き、純白と金の光の波が同心円状に広がる瞬間。光に触れた穢れた妖たちが、苦悶ではなく安らいだ表情になって淡い光の粒へ還っていく。飛び散る紙垂と花びら、ダイナミックな構図。激しいのに神聖で美しい、本作の戦闘の象徴。
```

### 4-2. 縛り祓い（結びの技）

```
ミトが放った朱色の注連縄が、光をまといながら扇状に広がり、巨大な妖を優しく縛り留める瞬間。縄の結び目が金色に輝く。躍動感のある斜め構図。「結び師の子」の技を象徴する一枚。
```

### 4-3. 岩鬼の王との対峙（体験版ボス）

```
苔と岩でできた巨大な山の守り神「岩鬼の王」が根の洞の最奥でミトと対峙する。体に黒紫の穢れが纏わりつくが、その奥の目はどこか哀しい。小さなミトが祓い刀を構えて立ち向かう、大きさの対比が際立つ構図。怖さより「祓ってあげたい」と感じる絵。
```

### 4-4. スニーク（探索の楽しさ）

```
格上の強大な妖が徘徊する谷を、ミトが岩陰に隠れて息を潜めてやり過ごす緊張の瞬間。視線の先には祠に置かれた輝く勾玉。妖の霊気が赤く揺らめき、ミトとシロが顔を見合わせる。緊張感の中にユーモアと冒険心がある絵。
```

## 5. キャラクターアート（主要キャラ・各2:3立ち絵推奨）

> 各プロンプト＝①共通スタイル＋以下。ミト同席の場合は②も貼る。

### 5-1. 廻人（めぐりと）— 父・結び師

```
40代の結び師の男性「廻人」。ミトの父。墨色の髪に白髪が混じり、短い髭、優しく芯の強い目元（ミトと似た琥珀色の瞳）。藍染の作務衣風の旅装に、組紐の束と道具を提げた革帯、使い込んだ手控え帳。背中に大きな旅嚢。風雪を経た旅人の風格と、子を想う柔らかさが同居する立ち絵。
```

### 5-2. 鈴代（すずしろ）— 育ての老巫女

```
小柄な老巫女「鈴代」。白髪を丸くまとめ、皺の深い笑顔。白い小袖に朱の袴、肩に古びた鈴のついた杖。霧立の里の鳥居の前で、温かくも見透かすような眼差し。厳しさと慈しみが同居する、物語の導き手の立ち絵。
```

### 5-3. シロ — 式鳥（父の使い）

```
白い折り紙でできた小鳥の精霊「シロ」。翼に淡く光る霊文（古い文字）が刻まれ、ふわりと空中に浮かぶ。少し欠けた羽根が健気さを感じさせる。可愛らしく、どこか切ない。デザインシート形式で、飛翔ポーズ・とまりポーズ・光の粒に還る演出の3パターン。
```

### 5-4. 常世の巫（とこよのかんなぎ）

```
「歌が人の形をとった」存在、常世の巫。白砂と波の色が透ける半透明の衣、海泡のような髪、目を閉じたまま微笑む顔。輪郭が波音とともに淡く揺らぎ、足元は砂に溶けている。人ではないが恐ろしくない、郷愁を誘う神秘の立ち絵。常世の浜の白珊瑚の鳥居を背景に。
```

### 5-5. 調（しらべ）— 天の古老の末席

```
高天原に残る天の古老「調」。年齢不詳の白金の髪、雲の文様の白い装束、静かで読めない表情の中に微かな悔いと優しさ。光の紐で編まれた数珠を手繰っている。神々しいが冷たくない、孤独な賢者の立ち絵。白金の廃都を背景に。
```

### 5-6. 岩鬼の王（体験版ボス）— 二態

```
山の守り神「岩鬼の王」のデザインシート。①穢れた姿: 苔むした巨岩の体に黒紫の穢れが纏わりつき、目に哀しみが宿る巨躯。②本来の姿: 穢れが祓われ、緑豊かな苔と花が体に咲く温和な山神。同じ体格・同じ顔立ちで「同一の存在」と分かるように2態を並べる。怖すぎない、どこか愛嬌のあるデザイン。
```

### 5-7. 天鎖守（あまくさりもり）— 白金の番人

```
天鎖が生み出した自動の番人「天鎖守」。白金の鎧のような体（中身は空洞で淡い光）、一つ目のような穏やかな光の核、体の各所に古い鎖と封印の文様。直立した姿は神像のように静謐で、悪意を感じさせない。高天原の意匠と連続する神聖なデザイン。デザインシート形式で立ち姿と戦闘姿勢の2態。
```
